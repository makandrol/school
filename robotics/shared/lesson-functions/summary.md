# Спільний урок: 🧩 Функції у MakeCode
**Для груп:** 5-АБ (урок 24, 31.03.2026), 5-ВГ (урок 7, 02.04.2026)
**Статус:** ✅ Створено

## Тема
Функції в MakeCode: створення, виклик, рефакторинг попередніх ігор.

## Що вивчили
- Що таке функція (аналогія: рецепт)
- Категорія «Розширені → Функції»
- Блоки: `функція назваФункції`, `викликати назваФункції`
- Рефакторинг: винесення повторюваного коду у функцію
- Функція може викликати іншу функцію

## Що зробили

### ⭐ Завдання 1 — Привіт з функцією:
- Функція `показатиПривіт`: показати ♥ → пауза 500 → 😊 → пауза 500 → очистити
- Викликається з: on button A, on button B, on shake
- Ключова ідея: зміна всередині функції — зміна скрізь

### ⭐⭐ Завдання 2 — Машинка з функцією перевіритиКордони:
- Рефакторинг гри «Машинка» з попереднього уроку
- Функція `перевіритиКордони`: if x > 4 → set x to 4, if x < 0 → set x to 0
- Кнопки A/B: change x + call перевіритиКордони
- Порівняння: до (повтор) vs після (функція)

### ⭐⭐⭐ Завдання 3 — Збирач палива з функціями:
- Функція `новеПаливо`: set паливоX = random 0-4, set паливоY = random 0-4
- Функція `перевіритиЗбір`: if x=паливоX and y=паливоY → score +1, call новеПаливо
- Демонстрація: функція може викликати іншу функцію

## MakeCode блоки
- `function`, `call function` (категорія Functions/Advanced)
- Всі попередні блоки: plot, unplot, clear screen, variables, if/else, buttons, etc.
