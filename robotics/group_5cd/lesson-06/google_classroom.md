### 📌 Урок 6. 🚗 Машинка на micro:bit — «Машинка», «Збирач палива» та «Не зачепи!»
**Дата:** 19.03.2026

#### 🎯 Що ми вивчили сьогодні:
- Повторили координати LED-екрану: x (0–4, зліва→направо), y (0–4, зверху→вниз)
- `plot x y` / `unplot x y` — запалити / погасити конкретний діод
- **Перевірка кордонів** — якщо x < 0 → задати x = 0, якщо x > 4 → задати x = 4 (аналогічно для y)
- Керування рухом діода кнопками A, B, A+B
- Перевірка зіткнення — чи два об'єкти на одному місці (блок `and` з Logic)
- Змінна-прапорець (`гра`) — для зупинки програми при Game Over

#### 💻 Що ми зробили:

**⭐ Завдання 1 — Машинка 🚗**
Машинка (один діод) їде по нижньому рядку (y = 4). Керуємо кнопками вліво-вправо.

| Кнопка | Дія |
|--------|-----|
| **A** | Вправо → (x + 1) |
| **B** | Вліво ← (x − 1) |

Змінна: `x`
- `при старті` → `задати x значення 2`
- `постійно` → `очистити екран` → `засвітити x (x) y 4` → `пауза 100`
- `коли кнопка A` → `змінити x на 1` → якщо x > 4 → задати x = 4
- `коли кнопка B` → `змінити x на −1` → якщо x < 0 → задати x = 0

**⭐⭐ Завдання 2 — Збирач палива ⛽**
Машинка їздить по всьому полю 5×5! На екрані з'являється «паливо» — збери 5 бочок!

| Кнопка | Дія |
|--------|-----|
| **A** | Рух вправо → (x + 1) |
| **B** | Рух вліво ← (x − 1) |
| **A+B** | Рух вниз ↓ (y + 1) |

Змінні: `x`, `y`, `паливоX`, `паливоY`, `бали`
- `при старті` → задати x=2, y=2, бали=0, паливоX та паливоY = випадкове 0–4
- `постійно` → очистити екран → засвітити машинку → засвітити паливо → перевірка зіткнення (якщо x = паливоX **і** y = паливоY → бали +1, нове паливо) → якщо бали = 5 → "WIN"
- Кнопки A/B — аналогічно Завданню 1, A+B — рух вниз з перевіркою кордонів

⚠️ Для перевірки двох умов одночасно використай блок `і (and)` з категорії **Logic**!

**⭐⭐⭐ Завдання 3 — Не зачепи! 🚧**
Перешкоди падають зверху вниз! Керуй машинкою і не зачепи!

| Елемент | Опис |
|---------|------|
| 🚗 Машинка | Нижній рядок (y=4), рухається вліво/вправо |
| 🚧 Перешкода | Починає з y=0, рухається вниз |
| 🏆 Бали | +1 за кожну перешкоду, яку оминув |

Змінні: `x`, `перешкодаX`, `перешкодаY`, `бали`, `гра`
- `при старті` → задати x=2, бали=0, гра=1, перешкодаX = випадкове 0–4, перешкодаY = 0
- `постійно` → якщо гра=1: очистити екран → засвітити машинку → засвітити перешкоду → перешкодаY +1 → якщо перешкодаY > 4 (оминув!) → бали+1, нова перешкода зверху → якщо перешкодаY=4 і перешкодаX=x → гра=0, показати "X" і бали → пауза 400
- Кнопки A/B — такі самі як у Завданні 1
