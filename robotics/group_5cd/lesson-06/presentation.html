🚗 Машинка на micro:bit

19.03.2026 | Група: 5-В, 5-Г

«Машинка» • «Збирач палива» • «Не зачепи!»

🔄 Повторення координат LED

Нагадаємо: LED-екран micro:bit — це сітка 5×5 діодів.

  • x — стовпець (0–4, зліва → направо)
  • y — рядок (0–4, зверху → вниз)

Блоки для роботи з діодами:

засвітити x __ y __ — запалити діод

погасити x __ y __ — погасити діод

💡 Сьогодні ключове: перевірка кордонів — щоб машинка не «вилетіла» за межі екрану!

x →

0,01,02,03,04,0 0,11,12,13,14,1 0,21,22,23,24,2 0,31,32,33,34,3 0,41,42,43,44,4

y ↓

🆕 Нове — перевірка кордонів

Коли машинка рухається кнопками, вона може «вилетіти» за край екрану!

Рішення — після кожного руху перевіряємо значення:

ПроблемаРішення
x стає менше 0якщо x < 0 → задати x значення 0
x стає більше 4якщо x > 4 → задати x значення 4
y стає менше 0якщо y < 0 → задати y значення 0
y стає більше 4якщо y > 4 → задати y значення 4
💡 Це як стіни кімнати — машинка впирається і далі не їде!
⚠️ Без цієї перевірки діод «зникне» за край і ти його не побачиш!

⭐ Завдання 1 — Машинка 🚗

Машинка їде по нижньому рядку (y = 4). Керуй кнопками!

КнопкаДія
AВправо → (x + 1)
BВліво ← (x − 1)
  • Машинка — це один діод на LED-екрані
  • Рухається тільки вліво-вправо
  • Не може вилізти за кордони! (0 ≤ x ≤ 4)

Змінна: x

Машинка на рядку y=4:

🚗

B: ← вліво   |   A: вправо →

⚠️ Запам'ятай: A — вправо (+1), B — вліво (−1)!

🛠️ Будуємо «Машинку» — блоки

Створи змінну: x

при старті:

задати x значення 2

постійно:

очистити екран

засвітити x x y 4

пауза 100 мс

коли кнопка A натиснута:

змінити x на 1

якщо x > 4

задати x значення 4

коли кнопка B натиснута:

змінити x на −1

якщо x < 0

задати x значення 0

💡 очистити екран + засвітити в циклі постійно — так машинка «рухається» (старий діод гасне, новий загорається).

⭐⭐ Завдання 2 — Збирач палива ⛽

Машинка їздить по всьому полю 5×5! На екрані з'являється «паливо» — збери його!

КнопкаДія
AРух вправо → (x + 1)
BРух вліво ← (x − 1)
A+BРух вниз ↓ (y + 1)
  • Машинка їде в 3 напрямках
  • Паливо — випадковий діод на екрані
  • Зібрав 5 бочок = перемога! 🏆
  • Кордони перевіряємо для x і y!

Машинка + паливо:

⛽ 🚗

Машинка (1,2) збирає паливо (3,0)

🏆 Хто збере 5 бочок найшвидше?

🛠️ Будуємо «Збирач палива» — блоки

Змінні: x, y, паливоX, паливоY, бали

при старті:

задати x значення 2

задати y значення 2

задати бали значення 0

задати паливоX значення випадкове від 0 до 4

задати паливоY значення випадкове від 0 до 4

постійно:

очистити екран

засвітити x x y y

засвітити x паливоX y паливоY

якщо x = паливоX і y = паливоY

змінити бали на 1

задати паливоX значення випадкове від 0 до 4

задати паливоY значення випадкове від 0 до 4

якщо бали = 5

показати рядок "WIN"

пауза 200 мс

коли кнопка A (вправо):

змінити x на 1

якщо x > 4 → задати x значення 4

коли кнопка B (вліво):

змінити x на −1

якщо x < 0 → задати x значення 0

коли кнопка A+B (вниз):

змінити y на 1

якщо y > 4 → задати y значення 4

⚠️ Для перевірки двох умов одночасно (x = паливоX і y = паливоY), використай блок і (and) з категорії Logic!

⭐⭐⭐ Завдання 3 — Не зачепи! 🚧

Перешкоди падають зверху вниз! Керуй машинкою і не зачепи!

ЕлементОпис
🚗 МашинкаНижній рядок (y=4), рухається вліво/вправо
🚧 ПерешкодаПочинає з y=0, рухається вниз
🏆 Бали+1 за кожну перешкоду, яку оминув
  • Перешкода — діод у випадковому стовпці
  • Рухається зверху вниз
  • Якщо торкнулась машинки — Game Over!

Змінні: x, перешкодаX, перешкодаY, бали, гра

Перешкода падає:

🚧 🚗

Не зачепи! A → вправо, B → вліво

💡 Це як справжня гра на телефоні — тільки на micro:bit!

🏁 Граємо та підсумки!

Міні-змагання:

  • 🚗 Машинка — покеруй 30 секунд, не вилазь за кордони!
  • ⛽ Збирач палива — хто збере 5 бочок найшвидше?
  • 🚧 Не зачепи! — хто набере найбільше балів?

Що ми сьогодні вивчили:

  • ✅ Перевірка кордонів — щоб об'єкт не виходив за межі 0–4
  • ✅ Керування рухом діода кнопками A, B, A+B
  • ✅ Зіткнення — перевірка, чи два об'єкти на одному місці
  • ✅ Змінна-прапорець (гра) — для зупинки програми
  • ✅ Створили 3 ігри! 🎮🎮🎮
🏠 Вдома: зроби перешкоди швидшими з часом (зменшуй pause кожні 5 балів), або додай рух вгору у «Збирач палива»!
