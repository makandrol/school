# Урок 6: 🚗 Машинка на micro:bit — «Машинка», «Збирач палива» та «Не зачепи!»
**Дата:** 19.03.2026
**Статус:** 📅 Запланований

## Тема
Керування рухом діода на LED-екрані за допомогою кнопок. Перевірка кордонів (boundary checking). Перевірка зіткнень двох об'єктів. Змінна-прапорець для стану гри. Три ігри зростаючої складності.

Адаптовано з уроку 23 групи 5-АБ (спрощено для менш досвідченої групи).

## Що вивчили
- Повторення координат LED: x (0–4, зліва→направо), y (0–4, зверху→вниз)
- `plot x y` / `unplot x y` — запалити/погасити конкретний діод
- Перевірка кордонів: `if x < 0 → set x to 0`, `if x > 4 → set x to 4` (аналогічно для y)
- Керування рухом діода кнопками A (вправо), B (вліво), A+B (вниз)
- Перевірка зіткнення: `if x = паливоX and y = паливоY` (блок `and` з Logic)
- Змінна-прапорець `гра` (1 = грає, 0 = Game Over) — для зупинки гри
- Патерн «очистити екран → засвітити → пауза» в циклі `постійно` для анімації руху

## Що зробили

### ⭐ Завдання 1 — Машинка 🚗
Машинка (один діод) на нижньому рядку (y=4), рухається вліво-вправо кнопками A/B.

**Блоки:**
- Variables: `set x to 2`, `change x by 1`, `change x by −1`
- Basic: `clear screen`, `pause 100`
- LED: `plot x (x) y 4`
- Logic: `if x > 4`, `if x < 0`
- Input: `on button A pressed`, `on button B pressed`

### ⭐⭐ Завдання 2 — Збирач палива ⛽
Машинка їздить по всьому полю (3 напрямки). Збирає випадково розміщене паливо. 5 зібраних = перемога.

**Блоки:**
- Variables: `x`, `y`, `паливоX`, `паливоY`, `бали`
- Basic: `clear screen`, `pause 200`, `show string "WIN"`
- LED: `plot x (x) y (y)`, `plot x (паливоX) y (паливоY)`
- Logic: `if x = паливоX and y = паливоY`, `if бали = 5`, перевірка кордонів для x і y
- Math: `pick random 0 to 4`
- Input: `on button A pressed`, `on button B pressed`, `on button A+B pressed`

### ⭐⭐⭐ Завдання 3 — Не зачепи! 🚧
Перешкоди падають зверху. Машинка на y=4, рухається вліво-вправо. Торкнулась перешкода = Game Over. +1 бал за кожну оминуту перешкоду.

**Блоки:**
- Variables: `x`, `перешкодаX`, `перешкодаY`, `бали`, `гра`
- Basic: `clear screen`, `pause 400`, `show string "X"`, `show number бали`
- LED: `plot x (x) y 4`, `plot x (перешкодаX) y (перешкодаY)`
- Logic: `if гра = 1`, `if перешкодаY > 4`, `if перешкодаY = 4 and перешкодаX = x`
- Math: `pick random 0 to 4`
- Input: `on button A pressed`, `on button B pressed`

## MakeCode блоки (усі)
- `plot x y`, `unplot x y`, `clear screen`, `pause`, `show string`, `show number`
- `on button A/B/A+B pressed`, `pick random 0 to 4`
- `if`, `and`, `set variable`, `change variable by`
- Цикл `forever` (постійно)

## Самостійна робота
- Зробити перешкоди швидшими з часом (зменшувати pause кожні 5 балів)
- Додати четвертий напрямок руху (вгору) у «Збирач палива» через on shake
- Додати звуки при зборі палива або Game Over
- Додати другу перешкоду одночасно (змінні `перешкода2X`, `перешкода2Y`)
