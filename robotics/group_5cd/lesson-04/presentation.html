⭐ Завдання 1 — Генератор фігур

Трусимо → micro:bit показує випадкову фігуру

Що будуємо:

  • Змінна рука
  • on shake → set рука to pick random 0 to 2
  • if рука = 0 → камінь (заповнений квадрат)
  • else if рука = 1 → ножиці (хрестик)
  • else → папір (рамка)

Таблиця фігур:

ЧислоФігураLED
0🪨 КаміньКвадрат
1✂️ НожиціХрестик
2📄 ПапірРамка

⭐⭐ Завдання 2 — Гра проти micro:bit!

Micro:bit — наш суперник 🤖. Змінні: бот, я, бот_готовий

КнопкаЩо відбувається
AБот обирає фігуру → показує ?
BМи обираємо фігуру → бачимо свою картинку. Без A → покаже "A!"
A+BХто виграв? → :) перемога / :( програш / = нічия
SHAKEСкидаємо все → нова гра! Показує "КНП"

Кнопка A (бот):

set бот to pick random 0 to 2
set бот_готовий to 1
show icon [?]

Кнопка B (ми):

if бот_готовий = 0 → show "A!"
else → set я to pick random 0 to 2
       show картинку (if/else if/else)

A+B (результат):

if я = бот       → "="  (нічия)
if я=0 AND бот=1 → ":)" (камінь > ножиці)
if я=1 AND бот=2 → ":)" (ножиці > папір)
if я=2 AND бот=0 → ":)" (папір > камінь)
else             → ":(" (бот виграв)

SHAKE (скидання):

set бот=-1, я=-1, бот_готовий=0
show string "КНП"
1 /
← → стрілки для навігації
