### 📌 Урок 5. 🏎️ Ігри на micro:bit — «Старт Ф1», «Вгадай число» та «Зворотний відлік»
**Дата:** 05.03.2026

#### 🎯 Що ми вивчили сьогодні:
- `running time (ms)` — вбудований «годинник» micro:bit, рахує мілісекунди (Input → ...more)
- `plot x y` — запалити один конкретний діод на LED-матриці (Led → plot)
- `show arrow` — показати стрілку ↑ або ↓ на LED (Basic → show arrow)
- `while` — цикл «поки» — повторює блоки поки умова виконується (Loops → while)
- Обмеження діапазону — як не дати змінній вийти за межі (наприклад, не менше 1, не більше 10)
- Вимірювання часу: різниця двох `running time` = скільки пройшло часу

#### 💻 Що ми зробили:

**⭐ Завдання 1 — Знайомство з часом ⏰**
Ознайомились із блоком `running time (ms)` — це вбудований таймер micro:bit, який рахує мілісекунди з моменту ввімкнення.

- `on button A` → `show number running time (ms)` — показує час у мілісекундах
- `on button B` → `show number running time (ms) / 1000` — показує час у секундах
- Натискаємо А кілька разів — число зростає!

**⭐⭐ Завдання 2 — Гра «Старт Ф1» 🏎️ (реакція)**
Як справжній старт у Формулі-1! 5 вогнів загораються по черзі на LED, потім гаснуть — натискай Б якнайшвидше!

| Крок | Кнопка | Що відбувається |
|------|--------|-----------------|
| 1 | **A** | 5 вогнів загораються по черзі 🔴🔴🔴🔴🔴 |
| 2 | — | Випадкова пауза 1-3 секунди... |
| 3 | — | Вогні гаснуть! 💨 СТАРТ! |
| 4 | **B** | Натискай якнайшвидше! Побачиш час в мс |

🏆 Нормальна реакція: **200-400 мс**. Менше 200 = суперпілот! 🦸

Змінні: `старт`, `реакція`
- `on button A` → `plot x 0 y 2` → `pause 500` → `plot x 1 y 2` → ... → 5 вогнів → `pause (random 1000-3000)` → `clear screen` → `set старт = running time (ms)`
- `on button B` → `set реакція = running time (ms) − старт` → `show number реакція`

**⭐⭐ Завдання 3 — Гра «Вгадай число» 🔢**
Micro:bit загадує число від 1 до 10. Ти вгадуєш!

| Кнопка | Дія |
|--------|-----|
| **A** | Збільшити число на 1 (▲) |
| **B** | Зменшити число на 1 (▼) |
| **SHAKE** | Перевірити! ↑ більше / ↓ менше / ✓ вгадав! |

Змінні: `загадане`, `спроба`
- `on start` → `set загадане = pick random 1 to 10` → `set спроба = 5` → `show number спроба`
- `on button A` → `change спроба by 1` → якщо > 10, ставимо 10 → `show number спроба`
- `on button B` → `change спроба by -1` → якщо < 1, ставимо 1 → `show number спроба`
- `on shake` → якщо вгадав → ":)" + "YES!" + нова гра; якщо менше → стрілка ↑; якщо більше → стрілка ↓

**⭐⭐⭐ Завдання 4 — Гра «Зворотний відлік» ⏱️**
Micro:bit показує зворотний відлік 10, 9, 8... Але на випадковому числі — ❌ КІНЕЦЬ! Натисни B щоб спробувати зупинити таймер вчасно.

| Кнопка | Дія |
|--------|-----|
| **A** | Запустити відлік! 10→0 |
| **B** | Зупинити таймер! |

❌ Відлік зупиняється на випадковому числі від 3 до 8 — ти не знаєш коли!

Змінні: `стоп`, `рахунок`, `зупинено`
- `on button A` → `set стоп = pick random 3 to 8` → `set рахунок = 10` → `while рахунок > 0`: показуємо число, чекаємо 400 мс, якщо `рахунок = стоп` → ❌ "END" → вихід
- `on button B` → `set рахунок = 0` (зупиняємо відлік) → якщо зупинено вчасно → ✓ "OK!"

🎮 Можна грати по черзі — передаєш одногрупнику, в кого зупиниться — програв!

Новий блок: `while` (цикл «поки») — знайдеш в **Loops → while**

#### 🏠 Самостійна робота (за бажанням):

🏆 **Рекорди реакції:** зіграй 5 разів і запиши найкращий час. Хто в класі швидший за 250 мс?

🔢 **Лічильник спроб:** додай змінну `спроби`, збільшуй при кожному SHAKE, покажи "Вгадав за X спроб!" при перемозі

📈 **Складніше:** зміни діапазон на 1-20 або 1-50 — гра стане набагато складнішою!

🎵 **Зі звуком:** додай мелодію при вгадуванні (`play melody` в категорії Music)

🔊 **Відлік зі звуком:** додай `play tone` з наростаючою частотою під час відліку — чим ближче до кінця, тим вище звук!

⏰ **Відлік-таймер:** замість чисел 10→0, зроби таймер на випадковий час 5-15 секунд — покажи ❤️ і чекай `pause (random 5000-15000)` → ❌. Передавай по колу!
