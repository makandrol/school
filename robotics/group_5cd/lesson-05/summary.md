# Урок 5: 🏎️ Ігри на micro:bit — «Старт Ф1», «Реакція», «Вгадай число» та «Зворотний відлік»
**Дата:** 05.03.2026
**Статус:** 📅 Запланований

## Тема
Вимірювання часу (`running time`), стрілки (`show arrow`), цикл `while`, обмеження діапазону змінних. Чотири ігри: знайомство з таймером, тест реакції (Ф1 старт), вгадай число, зворотний відлік.

## Що вивчили
- `running time (ms)` — вбудований таймер micro:bit (Input → ...more), повертає мілісекунди з моменту ввімкнення
- Вимірювання проміжку часу: різниця двох `running time` = скільки пройшло
- `show arrow North/South` — показ стрілок ↑↓ на LED
- `while` — цикл «поки» (Loops → while) — повторює блоки поки умова виконується
- Обмеження діапазону: `if спроба > 10 → set спроба = 10` (не виходити за межі)
- `clear screen` — очистити LED-матрицю
- `show leds` — малювання довільних патернів на LED (рядок за рядком)

## Що зробили

### ⭐ Завдання 1 — Знайомство з часом (running time):
- `on button A` → `show number running time (ms)` — показує скільки мілісекунд працює micro:bit
- `on button B` → `show number running time (ms) / 1000` — показує секунди
- Поекспериментувати: натиснути А кілька разів — число зростає!

### ⭐⭐ Завдання 2 — Гра «Старт Ф1» (реакція) 🏎️:
- Натискаємо А — починається старт: 5 діодів загораються по черзі (як у Формулі-1)
- Коли всі 5 гаснуть — натискай Б якнайшвидше!
- Micro:bit показує час реакції в мілісекундах
- `on button A` → запалюємо 5 LED по черзі (pause 500 між ними) → pause (random 1000-3000) → `clear screen` → `set старт = running time`
- `on button B` → `set реакція = running time − старт` → `show number реакція`
- Нормальна реакція: 200-400 мс

### ⭐⭐ Завдання 3 — Гра «Вгадай число»:
- Змінні: `загадане` (random 1-10), `спроба` (починаємо з 5)
- `on button A` → спроба +1 (max 10), `on button B` → спроба -1 (min 1)
- `on shake` → перевірка: вгадав (✓ + "YES!" + нова гра), менше (стрілка ↑), більше (стрілка ↓)
- Автоматичний перезапуск після вгадування

### ⭐⭐⭐ Завдання 4 — Гра «Зворотний відлік» ⏱️:
- Змінні: `стоп` (random 3-8), `рахунок` (починаємо з 10), `зупинено` (прапорець)
- `on button A` → запуск: `while рахунок > 0` — показуємо число, pause 400, якщо рахунок = стоп → ❌ "END"
- `on button B` → зупиняємо відлік (set рахунок = 0), якщо зупинено вчасно → ✓ "OK!"
- Використовує цикл `while` — новий блок!
- Можна грати по черзі: передавай micro:bit, в кого зупиниться — програв

## MakeCode блоки

## Самостійна робота
- Старт Ф1: зіграти 5 разів, записати найкращий результат
- Вгадай число: додати лічильник спроб
- Зворотний відлік зі звуком: play tone з наростаючою частотою
- Зворотний відлік-таймер: випадковий час 5-15 сек замість чисел
- Або: збільшити діапазон Вгадай число до 1-20
