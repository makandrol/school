#### 3. ⭐ Завдання 1 — Знайомство з часом (5 хв)
**Суть:** Ознайомитися з блоком `running time (ms)` — вбудованим таймером micro:bit.

**Алгоритм:**
```
on button A pressed:
  show number running time (ms)       ← показує мілісекунди з моменту ввімкнення

on button B pressed:
  show number running time (ms) / 1000  ← показує секунди
```

**Розбір:**
- `running time` — це "годинник" micro:bit, рахує мілісекунди з моменту ввімкнення
- 1 секунда = 1000 мілісекунд
- Натисни А кілька разів — число зростає! Бо micro:bit працює довше
- Це як секундомір, який ніколи не зупиняється
- Де знайти: **Input → ...more** (натисни «...more» внизу категорії Input!)

#### 4. ⭐⭐ Завдання 2 — Гра «Старт Ф1» (реакція) 🏎️ (12 хв)
**Суть:** Як у Формулі-1! На старті 5 вогнів загораються по черзі. Коли всі гаснуть — тисни Б! Micro:bit покаже час реакції.

**Алгоритм:**
```
on button A pressed:
  ← Загораємо 5 вогнів по черзі (як у Ф1):
  plot x 0 y 2                         ← перший вогонь
  pause 500
  plot x 1 y 2                         ← другий
  pause 500
  plot x 2 y 2                         ← третій
  pause 500
  plot x 3 y 2                         ← четвертий
  pause 500
  plot x 4 y 2                         ← п'ятий — всі горять!
  pause (pick random 1000 to 3000)     ← випадкова пауза 1-3 секунди
  clear screen                          ← ВОГНІ ПОГАСЛИ — СТАРТ!
  set старт = running time (ms)         ← запам'ятовуємо момент старту

on button B pressed:
  set реакція = running time (ms) − старт  ← різниця = час реакції
  show number реакція                  ← показуємо результат в мс!
```

**Змінні:** `старт` (момент гасіння вогнів), `реакція` (результат)

**Розбір:**
- `plot x y` — запалює один діод на координатах x, y (категорія Led)
- 5 діодів загораються по черзі — як 5 червоних вогнів на старті Ф1
- Після випадкової паузи всі гаснуть — це сигнал «СТАРТ!»
- Гравець натискає Б якнайшвидше
- Різниця двох `running time` = час реакції
- Нормальна реакція людини: 200-400 мс
- Менше 200 = супершвидко! Більше 500 = треба тренуватись

#### 5. ⭐⭐ Завдання 3 — Гра «Вгадай число» (10 хв)
**Суть:** Micro:bit загадує число від 1 до 10. Гравець вгадує — кнопка A збільшує спробу, B зменшує. Shake — перевірити. Micro:bit показує стрілки ↑ (більше) або ↓ (менше) або ✓ (вгадав!).

**Алгоритм:**
```
on start:
  set загадане = pick random 1 to 10
  set спроба = 5                       ← починаємо з середини
  show number спроба

on button A:
  change спроба by 1
  if спроба > 10 → set спроба = 10     ← не більше 10
  show number спроба

on button B:
  change спроба by -1
  if спроба < 1 → set спроба = 1       ← не менше 1
  show number спроба

on shake:
  if спроба = загадане:
    show icon ✓ (happy)
    show string "YES!"
    pause 2000
    set загадане = pick random 1 to 10  ← нова гра!
    set спроба = 5
    show number спроба
  else if спроба < загадане:
    show arrow North                    ← стрілка вгору = "більше!"
    pause 500
    show number спроба
  else:
    show arrow South                    ← стрілка вниз = "менше!"
    pause 500
    show number спроба
```

**Змінні:** `загадане` (таємне число), `спроба` (поточне число гравця)

#### 6. ⭐⭐⭐ Завдання 4 — Гра «Зворотний відлік» ⏱️ (6 хв)
**Суть:** Micro:bit показує зворотний відлік (10, 9, 8...), але зупиняється на випадковому числі — КІНЕЦЬ! 💀 Гравець натискає A, щоб запустити, а потім B, щоб «зупинити таймер» (спробувати встигнути). Якщо натиснув B до зупинки — врятувався, якщо ні — програв!

**Алгоритм:**
```
on button A:
  set стоп = pick random 3 to 8        ← зупиниться на цьому числі
  set рахунок = 10
  set зупинено = 0

  while рахунок > 0:
    show number рахунок
    pause 400
    if рахунок = стоп AND зупинено = 0:
      show icon 💀 (skull)               ← КІНЕЦЬ!
      set зупинено = 1
      show string "END"
      break                              ← вихід з циклу
    change рахунок by -1

  if зупинено = 0:                      ← дійшли до 0 без зупинки?
    show icon ✓
    show string "OK!"

on button B:
  if зупинено = 0:
    set рахунок = 0                     ← зупиняємо відлік!
```

**Альтернативний (простіший) варіант:**
```
on button A:
  set час = pick random 5000 to 15000  ← 5-15 секунд
  show icon ❤️                          ← тікає...
  pause час
  show icon 💀                          ← КІНЕЦЬ!
  show string "END!"
```
Цей варіант — просто таймер, в кого зупиниться — той програв. Передаємо по черзі!

**Змінні:** `стоп` (число зупинки), `рахунок` (поточний відлік), `зупинено` (прапорець)
