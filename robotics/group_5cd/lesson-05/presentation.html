🆕 Новий блок: running time

У micro:bit є вбудований «годинник» — рахує мілісекунди з моменту ввімкнення.

БлокЩо робитьДе знайти
running time (ms)Поточний час в мілісекундахInput → ...more
💡 1 секунда = 1000 мілісекунд. Якщо micro:bit увімкнений 5 секунд, running time = 5000!

Навіщо? Можна виміряти, скільки часу пройшло між двома моментами:

час_реакції = running time ЗАРАЗ − running time ТОДІ

⭐ Завдання 1 — Знайомство з часом ⏰

Спробуй вбудований таймер micro:bit!

Кнопка A — час у мілісекундах:

on button A:
  show number running time (ms)         ← скільки мс працює micro:bit

Кнопка B — час у секундах:

on button B:
  show number running time (ms) / 1000  ← переводимо в секунди
🧪 Натисни А кілька разів — число зростає! Бо micro:bit працює все довше. Це як секундомір, який ніколи не зупиняється!
⚠️ running time знайдеш: Input → ...more (натисни «...more» внизу категорії Input!)

⭐⭐ Завдання 2 — Гра «Старт Ф1» 🏎️

Як справжній старт у Формулі-1! 5 вогнів загораються → гаснуть → СТАРТ!

КрокКнопкаЩо відбувається
1A5 вогнів загораються по черзі 🔴🔴🔴🔴🔴
2—Випадкова пауза 1-3 секунди...
3—Всі вогні гаснуть! 💨 СТАРТ!
4BНатискай якнайшвидше! Побачиш час в мс
🏆 Нормальна реакція людини: 200-400 мс. Менше 200 = суперпілот! 🦸

🛠️ Будуємо «Старт Ф1»

Змінні: старт, реакція

Кнопка A — старт:

on button A:
  plot x 0 y 2                          ← 🔴 перший вогонь
  pause 500
  plot x 1 y 2                          ← 🔴🔴 другий
  pause 500
  plot x 2 y 2                          ← 🔴🔴🔴 третій
  pause 500
  plot x 3 y 2                          ← 🔴🔴🔴🔴 четвертий
  pause 500
  plot x 4 y 2                          ← 🔴🔴🔴🔴🔴 всі горять!
  pause (pick random 1000 to 3000)      ← пауза 1-3 сек...
  clear screen                           ← ВОГНІ ПОГАСЛИ = СТАРТ!
  set старт = running time (ms)          ← запам'ятали час

Кнопка B — результат:

on button B:
  set реакція = running time (ms) − старт
  show number реакція                    ← час в мілісекундах!
💡 plot x y — запалює один діод. Знайдеш: Led → plot x y

⭐⭐ Завдання 3 — Гра «Вгадай число» 🔢

Micro:bit загадав число від 1 до 10. Вгадай!

КнопкаДія
AЗбільшити число на 1 (▲)
BЗменшити число на 1 (▼)
SHAKEПеревірити! ↑ більше / ↓ менше / ✓ вгадав!
🎯 Підказка: якщо думати логічно, можна вгадати за 4-5 спроб максимум!

🛠️ Будуємо «Вгадай число»

Змінні: загадане, спроба

on start:

set загадане = pick random 1 to 10
set спроба = 5
show number спроба

Кнопки A / B:

on button A → change спроба by 1
               if спроба > 10 → set спроба = 10
               show number спроба

on button B → change спроба by −1
               if спроба < 1 → set спроба = 1
               show number спроба

🛠️ SHAKE — перевірка!

on shake:
  if спроба = загадане:
    show icon ✓ (happy face)       ← ВГАДАВ!
    show string "YES!"
    pause 2000
    ← НОВА ГРА:
    set загадане = pick random 1 to 10
    set спроба = 5
    show number спроба

  else if спроба < загадане:
    show arrow North               ← ↑ БІЛЬШЕ!
    pause 500
    show number спроба

  else:
    show arrow South               ← ↓ МЕНШЕ!
    pause 500
    show number спроба
💡 show arrow знайдеш в Basic → show arrow, вибери North (↑) або South (↓)

⭐⭐⭐ Завдання 4 — Гра «Зворотний відлік» ⏱️

Micro:bit рахує 10, 9, 8... але зупиняється на випадковому числі! ❌

КнопкаДія
AЗапустити відлік! 10→0
BСпробувати зупинити таймер вчасно!
❌ Відлік зупиняється на випадковому числі від 3 до 8. Ти не знаєш коли! Натисни B до зупинки!
🎮 Можна грати по черзі — передавай micro:bit одногрупнику! В кого зупиниться — той програв!

🛠️ Будуємо «Зворотний відлік»

Змінні: стоп, рахунок, зупинено

Кнопка A — запуск:

on button A:
  set стоп = pick random 3 to 8       ← коли зупиниться?
  set рахунок = 10
  set зупинено = 0
  while рахунок > 0:
    show number рахунок
    pause 400
    if рахунок = стоп:
      show icon ❌                      ← КІНЕЦЬ!
      show string "END"
      set зупинено = 1
      ← вихід з циклу (break)
    change рахунок by −1

Кнопка B — зупинити:

on button B:
  set рахунок = 0                      ← зупиняємо відлік!
  if зупинено = 0:
    show icon ✓                        ← Встиг!
    show string "OK!"
💡 while — цикл «поки». Знайдеш: Loops → while
