## Ключові концепції, які учні вже знають:
- Що таке micro:bit, LED матриця 5x5
- Вивід зображень та тексту на LED екран
- Анімація (послідовність зображень з паузами)
- Кнопки A, B, A+B — обробка подій
- Датчики: температура, освітлення, компас, нахил (акселерометр)
- Жест Shake (on shake) — запуск при струшуванні
- Змінні: створення, збільшення, відображення
- Випадкові числа: pick random 0 to N
- Цикли (forever, pause)
- Умовні оператори (if/else, if/else if/else)
- Вимірювання часу: running time (ms)
- Стрілки: show arrow North/South
- Цикл while (поки)
- Координати LED: plot x y, unplot x y (керування окремими діодами)
- Перевірка кордонів: if x > 4, if x < 0 (не вилазити за межі LED-матриці)
- Перевірка зіткнень: порівняння координат двох об'єктів (блок and)
- Змінна-прапорець (гра = 1/0) для зупинки forever
