# Урок 22: 🏎️ Формула-1 на micro:bit — «Старт Ф1», «Гонка» та «Піт-стоп»
**Дата:** 10.03.2026
**Статус:** 📅 Запланований

## Тема
Координати LED-матриці (блоки `plot x y`, `unplot x y`). Три проєкти у стилі Формули-1: стартові вогні горизонтально з вимірюванням реакції, гонка з керуванням швидкістю, піт-стоп з таймером.

## Що вивчили
- Блок `plot x y` — запалити окремий діод за координатами (Led)
- Блок `unplot x y` — погасити окремий діод (Led)
- Координати LED-матриці: x = 0-4 (зліва направо), y = 0-4 (зверху вниз)
- Закріплення: `running time`, `clear screen`, `pick random`, `forever`, змінні

## Що зробили

### ⭐ Завдання 1 — Світлофор Ф1 🚦:
- 5 вогнів загораються **горизонтально** зліва → направо по рядку y=2
- Блоки (on button A): `plot x 0 y 2` → `pause 1000` → `plot x 1 y 2` → ... → `plot x 4 y 2` → `pause (random 1000-4000)` → `clear screen` → `set старт to running time`
- Блоки (on button B): `set реакція to (running time − старт)` → `show number реакція`
- Реакція пілотів Ф1 ≈ 200 мс

### ⭐⭐ Завдання 2 — Гонка 🏁:
- Машинка (діод) рухається по рядку y=2
- Блоки (forever): `clear screen` → `plot x позиція y 2` → `pause швидкість` → `change позиція by 1`
- Коли `позиція > 4` → назад на 0, `коло +1`, показати номер кола
- Коли `коло = 5` → фініш, "P1"
- Кнопка A: `change швидкість by −50` (мін 50), B: `change швидкість by 50` (макс 500)

### ⭐⭐⭐ Завдання 3 — Піт-стоп ⏱️:
- 4 шини з'являються по черзі в кутах: (1,1), (3,1), (1,3), (3,3)
- Кожна з випадковою паузою 500-1500 мс + показ ✓
- В кінці — загальний час: `running time − старт`

## MakeCode блоки
- `plot x y`, `unplot x y`, `clear screen`
- `running time (ms)`, `pick random`, `show number`, `show icon`, `show string`
- `forever`, `pause`, `if/else`
- `change variable by`, `set variable`

## Самостійна робота
- Світлофор зі звуком: блок `play tone` при кожному вогні
- Гонка з перешкодами: випадкові діоди-перешкоди на трасі
- Таймер гонки: вимірювання загального часу 5 кіл
- Піт-стоп з кнопками: натискати B для заміни кожної шини
