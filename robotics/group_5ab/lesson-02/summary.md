# Урок 21: 🎯 Ігри на micro:bit — «Реакція», «Вгадай число» та «Зворотний відлік»
**Дата:** 03.03.2026
**Статус:** 📅 Запланований

## Тема
Вимірювання часу (`running time`), стрілки (`show arrow`), цикл `while`, обмеження діапазону змінних. Три ігри: тест реакції, вгадай число, зворотний відлік.

## Що вивчили
- `running time (ms)` — вбудований таймер micro:bit (Input → ...more), повертає мілісекунди з моменту ввімкнення
- Вимірювання проміжку часу: різниця двох `running time` = скільки пройшло
- `show arrow North/South` — показ стрілок ↑↓ на LED
- `while` — цикл «поки» (Loops → while) — повторює блоки поки умова виконується
- Обмеження діапазону: `if спроба > 10 → set спроба = 10` (не виходити за межі)
- `clear screen` — очистити LED-матрицю

## Що зробили

### ⭐ Завдання 1 — Гра «Реакція»:
- `on button A` → `clear screen` → `pause(random 1000-5000)` → `show icon ❤️` → `set старт = running time`
- `on button B` → `set реакція = running time − старт` → `show number реакція`
- Нормальна реакція: 200-400 мс

### ⭐⭐ Завдання 2 — Гра «Вгадай число»:
- Змінні: `загадане` (random 1-10), `спроба` (починаємо з 5)
- `on button A` → спроба +1 (max 10), `on button B` → спроба -1 (min 1)
- `on shake` → перевірка: вгадав (✓ + "YES!" + нова гра), менше (стрілка ↑), більше (стрілка ↓)
- Автоматичний перезапуск після вгадування

### ⭐⭐⭐ Завдання 3 — Гра «Зворотний відлік»:
- Змінні: `стоп` (random 3-8), `рахунок` (починаємо з 10), `програв` (прапорець)
- `on button A` → запуск: `while рахунок > 0` — показуємо число, pause 400, якщо рахунок = стоп → ❌ "СТОП"
- `on button B` → зупиняємо відлік (set рахунок = 0), якщо не програв → ✓ "OK!"
- Використовує цикл `while` — новий блок!
- Можна грати по черзі: передавай micro:bit, в кого зупиниться — програв

## MakeCode блоки
- `running time (ms)`, `clear screen`, `show arrow North/South`
- `while`, `pick random`, `show number`, `show icon`, `show string`
- `if/else if/else`, `change variable by`, `set variable`

## Самостійна робота
- Реакція: зіграти 5 разів, записати найкращий результат
- Вгадай число: додати лічильник спроб
- Зворотний відлік зі звуком: play tone з наростаючою частотою
- Секундний таймер: випадковий час 5-15 сек замість чисел
- Або: збільшити діапазон Вгадай число до 1-20
