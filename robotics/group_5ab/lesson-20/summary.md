# Урок 20: 🎮 Камінь-Ножиці-Папір
**Дата:** 24.02.2026
**Статус:** 📅 Запланований

## Тема
Випадкові числа, жест Shake, гра «Камінь-Ножиці-Папір» — від простого генератора до повноцінної гри проти micro:bit.

## Що вивчили
- Випадкові числа: блок `pick random 0 to N` (категорія Math)
- Жест Shake: блок `on shake` (категорія Input) — запуск при струшуванні
- Складна умовна конструкція: `if / else if / else` для кількох варіантів
- Зв'язок числа з дією: 0 → камінь, 1 → ножиці, 2 → папір
- Змінні-прапорці: використання змінної `бот_готовий` для контролю стану гри (чи зробив бот хід)
- Логіка перевірки перемоги: порівняння двох змінних за правилами КНП

## Що зробили

### ⭐ Завдання 1 — Генератор фігур (просте):
- `on shake` → генерується випадкове число 0–2
- `if рука = 0` → show leds (камінь — заповнений квадрат)
- `else if рука = 1` → show leds (ножиці — хрест-навхрест)
- `else` → show leds (папір — рамка)

### ⭐⭐ Завдання 2 — Гра проти micro:bit:
- Змінні: `бот`, `я`, `бот_готовий`
- `on start` → ініціалізація змінних, показати "КНП"
- `on button A` → бот обирає (pick random 0 to 2), ставимо `бот_готовий = 1`, показуємо «?»
- `on shake` → якщо `бот_готовий = 0` → show string "A!" (нагадування), інакше → генеруємо наш вибір і показуємо картинку
- `on button B` → порівнюємо `я` і `бот`: нічия "=", перемога ":)", програш ":("
- `on button A+B` → скидаємо все: бот=-1, я=-1, бот_готовий=0, показуємо "КНП"

## MakeCode блоки
- `on shake`, `on button A`, `on button B`, `on button A+B`
- `pick random 0 to 2`, `set variable`, `if/else if/else`
- `show leds`, `show string`, `show icon`

## Самостійна робота
- Доробити завдання 2 якщо не встигли
- Додати лічильник перемог (мої_перемоги, перемоги_бота)
- Або додати 4-ту фігуру «Колодязь» (pick random 0 to 3)
- Або додати звуки при результаті (play melody / play tone)
