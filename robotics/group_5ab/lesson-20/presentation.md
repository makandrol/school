## 🆕 Випадкові числа (Random)

Іноді програмі потрібно обрати щось **навмання** — як підкинути монетку!

- Блок `pick random 0 to 2` → дає число: **0**, **1** або **2**
- Кожен раз — інше! Ніколи не знаєш заздалегідь 🎲
- Де знайти: категорія **Math**

> 💡 Випадкові числа — це основа всіх комп'ютерних ігор! Кубики, карти, генерація монстрів — все це random!

---

## 🆕 Жест «Shake» (Струсити)

- Ми вже знаємо: `on button A`, `on tilt left`...
- Є ще: **`on shake`** — спрацьовує коли **струсити** micro:bit!
- В симуляторі: натисни кнопку **SHAKE** 🫨
- Де знайти: категорія **Input**

---

## Нові блоки MakeCode

| Блок | Що робить | Де знайти |
|------|-----------|-----------|
| `on shake` | Спрацьовує при струшуванні | **Input** |
| `pick random 0 to 2` | Випадкове число 0, 1 або 2 | **Math** |
| `if / else if / else` | Перевірка кількох умов | **Logic** (➕) |
| `on button A+B` | Обидві кнопки одночасно | **Input** |

> 💡 В блоці `if` натисни **➕** внизу, щоб додати `else if`!

---

## ⭐ Завдання 1 — Генератор фігур

**Просте:** Трусимо → micro:bit показує фігуру

| Число | Фігура | Картинка LED |
|-------|--------|-------------|
| 0 | 🪨 Камінь | Квадрат (кулак) |
| 1 | ✂️ Ножиці | Хрест-навхрест |
| 2 | 📄 Папір | Рамка (долоня) |

**Будуємо:**
1. Змінна `рука`
2. `on shake` → `set рука to pick random 0 to 2`
3. `if рука=0` → камінь, `else if рука=1` → ножиці, `else` → папір

---

## ⭐⭐ Завдання 2 — Гра проти micro:bit!

Граємо **ПРОТИ** плати! Micro:bit — наш суперник 🤖

| Крок | Кнопка | Що відбувається |
|------|--------|-----------------|
| 1 | **A** | Бот обирає (показує **?**) |
| 2 | **SHAKE** | Ми обираємо (бачимо свою фігуру) |
| 3 | **B** | Хто виграв? **:)** / **:(** / **=** |
| 4 | **A+B** | Скидаємо — нова гра! |

> ⚠️ Якщо SHAKE без A → micro:bit покаже **"A!"** — нагадає натиснути A!

---

## 🛠️ Завдання 2 — Змінні

Потрібні **3 змінні:**

| Змінна | Для чого |
|--------|----------|
| `бот` | Вибір micro:bit (0, 1 або 2) |
| `я` | Наш вибір (0, 1 або 2) |
| `бот_готовий` | Чи натиснули A? (0 = ні, 1 = так) |

```
on start → set бот = -1, я = -1, бот_готовий = 0
           show string "КНП"
```

---

## 🛠️ Завдання 2 — Кнопки A та SHAKE

### Кнопка A — хід бота:
```
on button A → set бот to pick random 0 to 2
              set бот_готовий to 1
              show icon [?]
```

### SHAKE — наш хід:
```
on shake →
  if бот_готовий = 0 → show string "A!"
  else → set я to pick random 0 to 2
         if я=0 → камінь, я=1 → ножиці, else → папір
```

---

## 🛠️ Завдання 2 — Кнопка Б і результат

### Кнопка Б — хто виграв?
```
on button B →
  if я = бот          → show "="   (нічия)
  if я=0 AND бот=1    → show ":)"  (камінь б'є ножиці)
  if я=1 AND бот=2    → show ":)"  (ножиці ріжуть папір)
  if я=2 AND бот=0    → show ":)"  (папір загортає камінь)
  else                → show ":("  (бот виграв)
```

### Кнопка A+Б — нова гра:
```
on button A+B → set бот=-1, я=-1, бот_готовий=0
                show string "КНП"
```

## 🎮 Граємо!

### Порядок гри:
1. **A** → бачимо **?** (бот обрав!)
2. **SHAKE** → бачимо свою фігуру
3. **B** → результат: **:)** / **:(** / **=**
4. **A+B** → нова гра!
