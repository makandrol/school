### Хід уроку:

#### 1. Вступ (3–5 хв)
- Привітання
- Інструктаж з БЖД
- Запитання: «Хто грав у камінь-ножиці-папір? А що, якщо micro:bit зможе грати з вами?»
- Сьогодні ми створимо гру, де micro:bit — це ваш суперник!

#### 2. Теоретична частина (8–10 хв)

**Нове поняття — Випадкові числа (Random):**
- Іноді програмі потрібно «підкинути монетку» — обрати щось навмання
- Блок `pick random 0 to [N]` — генерує випадкове число від 0 до N
- Приклад: `pick random 0 to 2` → дасть 0, 1, або 2
- Це як кидати кубик, тільки програмно!

**Нове поняття — Жест «Shake» (Струсити):**
- Ми вже знаємо `on button A pressed`, `on tilt left` тощо
- Є ще один крутий тригер: `on shake` — спрацьовує коли струсити micro:bit
- В симуляторі: натиснути кнопку «SHAKE» на екрані

**Демонстрація:** відкриваємо MakeCode, показуємо:
- Категорія **Input** → `on shake`
- Категорія **Math** → `pick random 0 to 2`
- Категорія **Logic** → `if ... then ... else if`

---

#### 3. Практична частина — Завдання 1 (12–15 хв)

##### ⭐ Завдання 1: Генератор фігур (просте)

**Ідея:** Трусимо micro:bit → він випадково показує камінь, ножиці або папір.

##### Крок 1. Створюємо змінну (2 хв)
- Категорія **Variables** → «Make a Variable» → назвемо `рука`

##### Крок 2. Генеруємо випадковий вибір при струшуванні (3 хв)
```
on shake:
    set "рука" to pick random 0 to 2
```
- 0 = камінь 🪨
- 1 = ножиці ✂️
- 2 = папір 📄

##### Крок 3. Показуємо картинку залежно від числа (5–7 хв)
```
on shake:
    set "рука" to pick random 0 to 2

    if "рука" = 0 then:
        show leds:            ← КАМІНЬ (квадрат/кулак)
            . # # # .
            # # # # #
            # # # # #
            # # # # #
            . # # # .

    else if "рука" = 1 then:
        show leds:            ← НОЖИЦІ
            # # . . #
            # # . # .
            . . # . .
            # # . # .
            # # . . #

    else:
        show leds:            ← ПАПІР (розкрита долоня)
            # # # # #
            # . . . #
            # . . . #
            # . . . #
            # # # # #
```

##### Крок 4. Тестуємо! (2–3 хв)
- Натискаємо SHAKE на симуляторі
- Перевіряємо: чи показує різні картинки?
- Грають один з одним — але незручно, бо кожен бачить тільки свій результат...

**Перехід:** «А що, якщо ви граєте самі вдома? З ким грати? Давайте навчимо micro:bit бути суперником!»

---

#### 4. Практична частина — Завдання 2 (12–15 хв)

##### ⭐⭐ Завдання 2: Гра проти micro:bit

**Ідея:** Граємо ПРОТИ плати! Micro:bit — наш суперник.

**Правила гри:**
1. Натискаємо **кнопку А** → micro:bit генерує СВІЙ вибір (але показує **?** — щоб ми не підглядали!)
2. Натискаємо **SHAKE** → micro:bit генерує НАШ вибір і показує його на екрані
3. Натискаємо **кнопку Б** → micro:bit порівнює і показує хто виграв!
4. Натискаємо **А+Б** → скидає все, грає нова гра
5. Якщо натиснути SHAKE без A — показує нагадування натиснути А

##### Крок 1. Створюємо змінні (2 хв)
- `бот` — вибір micro:bit (0, 1 або 2)
- `я` — наш вибір (0, 1 або 2)
- `бот_готовий` — чи натиснув гравець кнопку А (0 = ні, 1 = так)

```
on start:
    set "бот" to -1
    set "я" to -1
    set "бот_готовий" to 0
    show string "КНП"
```

##### Крок 2. Кнопка А — бот робить хід (3 хв)
```
on button A:
    set "бот" to pick random 0 to 2
    set "бот_готовий" to 1
    show icon [?]              ← показуємо знак питання
```
> Пояснення: бот вже обрав (число збережено в змінній `бот`), але ми бачимо тільки «?» — щоб не підглядати!

##### Крок 3. SHAKE — наш хід (4 хв)
```
on shake:
    if "бот_готовий" = 0 then:
        show string "A!"       ← нагадування: спочатку натисни А!
    else:
        set "я" to pick random 0 to 2

        if "я" = 0 then:
            show leds: (камінь)
        else if "я" = 1 then:
            show leds: (ножиці)
        else:
            show leds: (папір)
```
> Пояснення: якщо забув натиснути А — micro:bit нагадає! Інакше — генерує наш вибір.

##### Крок 4. Кнопка Б — хто виграв? (5 хв)

**Правила КНП:**
- Камінь (0) б'є ножиці (1)
- Ножиці (1) ріжуть папір (2)
- Папір (2) загортає камінь (0)
- Якщо однаково → нічия

```
on button B:
    if "я" = "бот" then:
        show string "="        ← нічия
    else if ("я"=0 AND "бот"=1) OR ("я"=1 AND "бот"=2) OR ("я"=2 AND "бот"=0) then:
        show string ":)"       ← ти виграв!
    else:
        show string ":("       ← бот виграв
```

> 💡 Підказка для спрощення: можна перевірити перемогу гравця окремими if:
```
on button B:
    if "я" = "бот" then:
        show string "="
    else if "я"=0 AND "бот"=1 then:
        show string ":)"
    else if "я"=1 AND "бот"=2 then:
        show string ":)"
    else if "я"=2 AND "бот"=0 then:
        show string ":)"
    else:
        show string ":("
```

##### Крок 5. А+Б — скидаємо гру (1 хв)
```
on button A+B:
    set "бот" to -1
    set "я" to -1
    set "бот_готовий" to 0
    show string "КНП"
```

##### Крок 6. Тестуємо! (3 хв)
- А → бачимо «?»
- SHAKE → бачимо свою фігуру
- Б → бачимо результат!
- А+Б → скидаємо, грає нова гра
- Перевіряємо: SHAKE без А → має показати «A!»

### Очікувані результати:
- Учень розуміє: що таке випадкові числа, навіщо вони потрібні в іграх
- Учень вміє: використовувати `pick random`, `on shake`, будувати складну конструкцію `if / else if / else`
- Учень розуміє: як використовувати змінні як «прапорці» для контролю стану програми

### Код проєкту (MakeCode blocks опис):

**Нові блоки:**
| Блок | Що робить | Де знайти |
|------|-----------|-----------|
| `on shake` | Спрацьовує при струшуванні | **Input** → on shake |
| `pick random 0 to 2` | Випадкове число від 0 до 2 | **Math** → pick random |
| `if / else if / else` | Перевірка кількох умов | **Logic** → if, натиснути ➕ щоб додати else if |
| `on button A+B` | Спрацьовує при одночасному натисканні A і B | **Input** → on button A+B |

**Логіка проєкту (завдання 2 — повна версія):**
1. `on start` → ініціалізація змінних `бот`, `я`, `бот_готовий`; показати "КНП"
2. `on button A` → `set бот to pick random 0 to 2`, `set бот_готовий to 1`, показати «?»
3. `on shake` → якщо `бот_готовий = 0` → показати "A!", інакше → `set я to pick random 0 to 2`, показати картинку
4. `on button B` → порівняти `я` і `бот`, показати "=", ":)" або ":("
5. `on button A+B` → скинути все, показати "КНП"
