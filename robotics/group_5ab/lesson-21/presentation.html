🆕 Новий блок: running time

У micro:bit є вбудований «годинник» — рахує мілісекунди з моменту ввімкнення.

БлокЩо робитьДе знайти
running time (ms)Поточний час в мілісекундахInput → ...more
💡 1 секунда = 1000 мілісекунд. Якщо micro:bit увімкнений 5 секунд, running time = 5000!

Навіщо? Можна виміряти, скільки часу пройшло між двома моментами:

час_реакції = running time ЗАРАЗ − running time ТОДІ

⭐ Завдання 1 — Гра «Реакція» ⏱️

Перевір свою швидкість реакції!

КрокКнопкаЩо відбувається
1AЕкран гасне... чекаємо... ⏳
2—Через 1-5 секунд з'являється ❤️
3BНатискай якнайшвидше!
4—Micro:bit показує час в мс
🏆 Нормальна реакція людини: 200-400 мс. Менше 200 = суперлюдина! 🦸

🛠️ Будуємо «Реакцію»

Змінні: старт, реакція

Кнопка A — старт:

on button A:
  clear screen                          ← гасимо екран
  pause (pick random 1000 to 5000)      ← чекаємо 1-5 секунд
  show icon ❤️                          ← ЗАРАЗ!
  set старт = running time (ms)         ← запам'ятали час

Кнопка B — стоп:

on button B:
  set реакція = running time (ms) − старт
  show number реакція                   ← час в мілісекундах!
⚠️ running time знайдеш: Input → ...more (натисни «...more» внизу категорії Input!)

⭐⭐ Завдання 2 — Гра «Вгадай число» 🔢

Micro:bit загадав число від 1 до 10. Вгадай!

КнопкаДія
AЗбільшити число на 1 (▲)
BЗменшити число на 1 (▼)
SHAKEПеревірити! ↑ більше / ↓ менше / ✓ вгадав!
🎯 Підказка: якщо думати логічно, можна вгадати за 4-5 спроб максимум!

🛠️ Будуємо «Вгадай число»

Змінні: загадане, спроба

on start:

set загадане = pick random 1 to 10
set спроба = 5
show number спроба

Кнопки A / B:

on button A → change спроба by 1
               if спроба > 10 → set спроба = 10
               show number спроба

on button B → change спроба by −1
               if спроба < 1 → set спроба = 1
               show number спроба

🛠️ SHAKE — перевірка!

on shake:
  if спроба = загадане:
    show icon ✓ (happy face)       ← ВГАДАВ!
    show string "YES!"
    pause 2000
    ← НОВА ГРА:
    set загадане = pick random 1 to 10
    set спроба = 5
    show number спроба

  else if спроба < загадане:
    show arrow North               ← ↑ БІЛЬШЕ!
    pause 500
    show number спроба

  else:
    show arrow South               ← ↓ МЕНШЕ!
    pause 500
    show number спроба
💡 show arrow знайдеш в Basic → show arrow, вибери North (↑) або South (↓)

⭐⭐⭐ Завдання 3 — Гра «Бомба» 💣

Micro:bit — це бомба! Рахує 10, 9, 8... але вибухає на випадковому числі! 💀

КнопкаДія
AЗапустити бомбу! Починається відлік 10→0
BСпробувати «розмінувати» (зупинити таймер)
💀 Бомба вибухає на випадковому числі від 3 до 8. Ти не знаєш коли! Натисни B до вибуху!
🎮 Можна грати по черзі — передавай micro:bit одногрупнику! В кого вибухне — той програв!

🛠️ Будуємо «Бомбу»

Змінні: бомба, рахунок, вибухнула

Кнопка A — запуск:

on button A:
  set бомба = pick random 3 to 8     ← коли вибухне?
  set рахунок = 10
  set вибухнула = 0
  while рахунок > 0:
    show number рахунок
    pause 400
    if рахунок = бомба:
      show icon 💀                    ← БУУУМ!
      show string "BOOM"
      set вибухнула = 1
      ← вихід з циклу (break)
    change рахунок by −1

Кнопка B — розмінувати:

on button B:
  set рахунок = 0                    ← зупиняємо відлік!
  if вибухнула = 0:
    show icon ✓                      ← Врятувався!
    show string "OK!"
💡 while — цикл «поки». Знайдеш: Loops → while

🎮 Граємо!

Змагання «Реакція»:

  • Кожен грає 3 рази → записує найкращий час
