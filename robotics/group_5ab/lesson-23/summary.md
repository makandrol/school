# Урок 23: 🚗 Машинка на micro:bit — «Машинка», «Збирач палива» та «Не зачепи!»
**Дата:** 17.03.2026
**Статус:** ✅ Створено

## Тема
Керування діодом кнопками A/B на LED-матриці з перевіркою кордонів. Три проєкти: машинка з рухом вліво-вправо, збирач палива на полі 5×5, гра з перешкодами.

## Що вивчили
- Перевірка кордонів: `if x > 4` → `set x to 4`, `if x < 0` → `set x to 0` (аналогічно для y)
- Керування рухом діода кнопками A, B, A+B
- Перевірка зіткнень двох об'єктів: `if x = паливоX and y = паливоY` (блок `and` з Logic)
- Змінна-прапорець (`гра` = 1/0) для зупинки forever при Game Over

## Що зробили

### ⭐ Завдання 1 — Машинка 🚗:
- Машинка (діод) на нижньому рядку y=4, рухається кнопками A (вправо) / B (вліво)
- Змінна `x` починає з 2
- Блоки (on start): `set x to 2`
- Блоки (forever): `clear screen` → `plot x (x) y 4` → `pause 100`
- Блоки (on button A): `change x by 1` → `if x > 4` → `set x to 4`
- Блоки (on button B): `change x by −1` → `if x < 0` → `set x to 0`

### ⭐⭐ Завдання 2 — Збирач палива ⛽:
- Машинка їздить по полю 5×5, кнопки A (вправо), B (вліво), A+B (вниз)
- Паливо — випадковий діод (`паливоX`, `паливоY`)
- Перевірка збігу координат: `if x = паливоX and y = паливоY` → `бали + 1`, нове паливо
- Зібрав 5 бочок → `show string "WIN"`
- Кордони перевіряються для x і y

### ⭐⭐⭐ Завдання 3 — Не зачепи! 💥:
- Перешкода рухається зверху вниз (y від 0 до 4)
- Машинка на y=4, керування A/B (як у Завданні 1)
- Перешкода пролетіла вниз → `бали + 1`, нова перешкода (новий випадковий x, y=0)
- Перешкода торкнулась машинки (`перешкодаY = 4 and перешкодаX = x`) → Game Over: показує "X" і бали
- Змінна `гра` (1/0) — прапорець для зупинки forever

## MakeCode блоки
- `plot x y`, `unplot x y`, `clear screen`
- `pick random`, `show number`, `show string`
- `forever`, `pause`, `if/else`, `and`
- `change variable by`, `set variable`

## Самостійна робота
- Не зачепи з прискоренням: зменшуй `pause` кожні 5 балів
- Кілька перешкод одночасно: додаткові змінні `перешкода2X`, `перешкода2Y`
- Збирач з таймером: вимірювання часу збирання 5 бочок через `running time`
