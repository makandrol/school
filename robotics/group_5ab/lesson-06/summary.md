# Урок 25: 🎮 Функції у грі — закріплення
**Дата:** 07.04.2026
**Статус:** ✅ Створено

## Тема
Закріплення роботи з функціями через створення 4 ігор. Кожна гра використовує функції для організації коду: від простого виклику з різних подій до складних ігор з кількома функціями, що викликають одна одну.

## Що закріпили
- Створення та виклик функцій (категорія Розширені → Функції)
- Одна функція — багато місць виклику
- Функція може викликати іншу функцію
- Організація ігрового коду через функції: намалювати, перевіритиКордони, перевіритиЗіткнення тощо

## Що зробили

### ⭐ Завдання 1 — Гра «Кубик емоцій» 🎲:
- Функція `показатиЕмоцію`: випадкове число 0–4 → if/else if → показує різні іконки (♥, 😊, 😢, 😮, 🦆) → пауза → очистити
- Викликається з: on shake, on button A, on button B
- Ключова ідея: одна функція, три точки виклику

### ⭐⭐ Завдання 2 — Гра «Влучний стрілець» 🎯:
- Функції: `новаМішень` (випадкові координати), `перевіритиКордони` (x 0–4), `вистрілити` (перевірка влучення → бали або -1 життя), `намалювати` (clear + plot)
- Керування: A вліво, B вправо, A+B вистріл
- 3 життя, Game Over при 0 життів
- Функція `вистрілити` викликає `новаМішень`

### ⭐⭐⭐ Завдання 3 — Гра «Космічний дощ» 🌧️:
- Краплі падають зверху вниз, гравець ловить внизу
- Функції: `новаКрапля` (random x, y=0), `перевіритиКордони`, `перевіритиЗбір` (перевірка координат), `намалювати`
- Кожні 5 балів — швидкість зростає (зменшення pause)
- Гра обмежена 30 секундами (running time)

### ⭐⭐⭐⭐ Завдання 4 — Гра «Пінг-понг» 🏓:
- М'яч літає по екрану, ракетка (3 діоди) внизу
- Функції: `рухМяча` (зміна координат, відбиття від стін), `перевіритиВідбиття` (м'яч на y=4 → перевірка ракетки), `намалювати` (clear + plot ball + plot paddle), `перевіритиКордони` (ракетка 1–3)
- Пропустив м'яч → Game Over

## MakeCode блоки
- `function`, `call function` (Functions/Advanced)
- `plot x y`, `unplot x y`, `clear screen` (LED)
- `pick random`, `show number`, `show string`, `show icon` (Basic/Math)
- `forever`, `pause`, `if/else`, `and`, `or` (Logic/Loops)
- `running time (ms)` (Input/more)
- Всі попередні блоки: buttons, variables, etc.

## Самостійна робота
- Додати анімацію Game Over через окрему функцію
- Додати рівні складності — функція `наступнийРівень`
- Переробити будь-яку попередню гру з використанням функцій
