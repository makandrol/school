#### 3. ⭐ Завдання 1 — «Світлофор Ф1» (стартові вогні) 🚦 (12 хв)
**Суть:** Як у Формулі-1! На старті 5 червоних вогнів загораються по черзі **зліва направо** (горизонтально, рядок y=2). Коли всі горять — випадкова пауза — і всі гаснуть! Це сигнал СТАРТ! Натисни Б якнайшвидше → micro:bit покаже час реакції.

**Блоки для `on button A pressed`:**
1. **Led → plot x 0 y 2** — перший вогонь (зліва)
2. **Basic → pause 1000**
3. **Led → plot x 1 y 2** — другий вогонь
4. **Basic → pause 1000**
5. **Led → plot x 2 y 2** — третій
6. **Basic → pause 1000**
7. **Led → plot x 3 y 2** — четвертий
8. **Basic → pause 1000**
9. **Led → plot x 4 y 2** — п'ятий — всі горять!
10. **Basic → pause** ← вставити **Math → pick random 1000 to 4000**
11. **Basic → clear screen** — ВОГНІ ПОГАСЛИ — СТАРТ!
12. **Variables → set `старт` to** ← вставити **Input → ...more → running time (ms)**

**Блоки для `on button B pressed`:**
1. **Variables → set `реакція` to** ← вставити **Math → 0 − 0**, де перше = **running time (ms)**, друге = **`старт`**
2. **Basic → show number `реакція`**

**Змінні:** `старт`, `реакція`

**Розбір:**
- **plot x y** (категорія **Led**) — запалює один діод на координатах x, y
- Координати LED: x = 0-4 (зліва направо), y = 0-4 (зверху вниз)
- 5 діодів загораються по черзі зліва направо по рядку y=2 — як 5 червоних панелей на старті Ф1
- Горизонтальне розташування — саме так виглядає справжній стартовий світлофор Ф1!
- Після випадкової паузи всі гаснуть — це сигнал «СТАРТ!»
- Різниця двох `running time` = час реакції
- У справжніх пілотів Ф1 реакція ~0.2 секунди (200 мс)!

#### 4. ⭐⭐ Завдання 2 — «Гонка» (LED-анімація) 🏁 (12 хв)
**Суть:** Автомобіль (один діод) їде по «трасі» — рухається зліва направо по LED-матриці. Кнопка A — прискорити, кнопка B — гальмувати. Мета — проїхати 5 кіл якнайшвидше!

**Блоки для `on start`:**
1. **Variables → set `швидкість` to 300**
2. **Variables → set `коло` to 0**
3. **Variables → set `позиція` to 0**

**Блоки для `forever`:**
1. **Basic → clear screen**
2. **Led → plot x `позиція` y 2** — малюємо машинку (змінну `позиція` перетягнути в поле x)
3. **Basic → pause `швидкість`** — (змінну `швидкість` перетягнути в поле)
4. **Variables → change `позиція` by 1**
5. **Logic → if `позиція` > 4** — доїхали до краю?
   - **Variables → set `позиція` to 0**
   - **Variables → change `коло` by 1**
   - **Basic → show number `коло`**
   - **Basic → pause 300**
6. **Logic → if `коло` = 5** — фініш!
   - **Basic → show string "P1"**

**Блоки для `on button A pressed`:**
1. **Variables → change `швидкість` by −50** — швидше!
2. **Logic → if `швидкість` < 50** → **Variables → set `швидкість` to 50**

**Блоки для `on button B pressed`:**
1. **Variables → change `швидкість` by 50** — повільніше
2. **Logic → if `швидкість` > 500** → **Variables → set `швидкість` to 500**

**Змінні:** `швидкість`, `коло`, `позиція`

**Розбір:**
- Машинка — один діод, який рухається вправо по рядку y=2
- Коли діод виходить за межу (x > 4), це = 1 коло пройдено
- Кнопка A — прискорення (менша пауза), B — гальмування (більша пауза)
- Після 5 кіл — фініш!

#### 5. ⭐⭐⭐ Завдання 3 — «Піт-стоп» ⏱️ (9 хв)
**Суть:** Симуляція піт-стопу! На екрані по черзі з'являються 4 «шини» (по одному діоду в кожному куті). Micro:bit рахує загальний час.

**Блоки для `on button A pressed`:**
1. **Variables → set `старт` to** ← **running time (ms)**
2. Шина 1:
   - **Basic → clear screen**
   - **Led → plot x 1 y 1** — зліва зверху
   - **Basic → pause** ← **pick random 500 to 1500**
   - **Basic → show icon ✓**
   - **Basic → pause 200**
3. Шина 2:
   - **Basic → clear screen**
   - **Led → plot x 3 y 1** — справа зверху
   - **Basic → pause** ← **pick random 500 to 1500**
   - **Basic → show icon ✓**
   - **Basic → pause 200**
4. Шина 3:
   - **Basic → clear screen**
   - **Led → plot x 1 y 3** — зліва знизу
   - **Basic → pause** ← **pick random 500 to 1500**
   - **Basic → show icon ✓**
   - **Basic → pause 200**
5. Шина 4:
   - **Basic → clear screen**
   - **Led → plot x 3 y 3** — справа знизу
   - **Basic → pause** ← **pick random 500 to 1500**
   - **Basic → show icon ✓**
   - **Basic → pause 200**
6. Результат:
   - **Variables → set `результат` to** ← **running time (ms) − `старт`**
   - **Basic → show number `результат`**

**Змінні:** `старт`, `результат`

**Розбір:**
- У Ф1 піт-стоп — це заміна 4 шин
- Найшвидший піт-стоп у Ф1 — ~1.8 секунди (Red Bull, 2019)!
- 4 шини з'являються по черзі в кутах з випадковою паузою
- Після 4 шин — micro:bit показує загальний час
