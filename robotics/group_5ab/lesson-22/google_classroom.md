### 📌 Урок 22. 🏎️ Формула-1 на micro:bit — «Старт Ф1», «Гонка» та «Піт-стоп»
**Дата:** 10.03.2026

#### 🎯 Що ми вивчили сьогодні:
- Блок `plot x y` — запалити окремий діод за координатами (категорія **Led**)
- Блок `unplot x y` — погасити окремий діод (категорія **Led**)
- Координати LED-матриці: x = 0-4 (зліва направо), y = 0-4 (зверху вниз)
- Закріпили: `running time`, `clear screen`, `pick random`, змінні

#### 💻 Що ми зробили:

**⭐ Завдання 1 — Світлофор Ф1 🚦**
Як на справжньому старті Формули-1! 5 червоних вогнів загораються по черзі **зліва направо** (горизонтально, рядок y=2). Коли всі горять — випадкова пауза 1-4 секунди — і ВСІ ГАСНУТЬ! Це сигнал СТАРТ! Натискай B якнайшвидше — micro:bit покаже час реакції.

| Крок | Що відбувається |
|------|-----------------|
| 1 | Натисни **A** — починається |
| 2 | 5 вогнів загораються зліва → направо (пауза 1 сек між ними) |
| 3 | Випадкова пауза... всі гаснуть — **СТАРТ!** |
| 4 | Натисни **B** якнайшвидше! Побачиш час в мс |

🏎️ Реакція пілотів Ф1 — приблизно 200 мс!

Блоки (on button A): `plot x 0 y 2` → `pause 1000` → `plot x 1 y 2` → `pause 1000` → `plot x 2 y 2` → `pause 1000` → `plot x 3 y 2` → `pause 1000` → `plot x 4 y 2` → `pause (pick random 1000 to 4000)` → `clear screen` → `set старт to running time (ms)`

Блоки (on button B): `set реакція to (running time − старт)` → `show number реакція`

**⭐⭐ Завдання 2 — Гонка 🏁**
Машинка (один діод) їде по «трасі» — рухається зліва направо по рядку y=2.

| Кнопка | Дія |
|--------|-----|
| **A** | Прискорити! 🏎️💨 (зменшує паузу на 50) |
| **B** | Гальмувати! 🛑 (збільшує паузу на 50) |

Змінні: `швидкість` (починаємо з 300), `коло` (0), `позиція` (0)

Блоки (forever): `clear screen` → `plot x позиція y 2` → `pause швидкість` → `change позиція by 1` → якщо `позиція > 4` → `set позиція to 0`, `change коло by 1`, `show number коло` → якщо `коло = 5` → `show string "P1"`

**⭐⭐⭐ Завдання 3 — Піт-стоп ⏱️**
Заміна 4 шин! Кожна шина з'являється у своєму куті LED-матриці з випадковою паузою.

4 позиції шин: (1,1), (3,1), (1,3), (3,3) — як 4 колеса машини!

🏎️ Рекорд піт-стопу в Ф1 — 1.8 секунди!

Блоки (on button A): `set старт to running time` → для кожної шини: `clear screen` → `plot x _ y _` → `pause (random 500-1500)` → `show icon ✓` → `pause 200` → після 4 шин: `set результат to (running time − старт)` → `show number результат`

#### 🏠 Самостійна робота (за бажанням):

🚦 **Світлофор зі звуком:** додай блок `play tone` (категорія **Music**) при кожному загорянні вогню!

🏁 **Перешкоди на трасі:** додай випадкові діоди-перешкоди — якщо машинка «зіткнулась» → game over!

⏱️ **Таймер гонки:** виміряй загальний час 5 кіл за допомогою `running time`

🔧 **Піт-стоп з кнопками:** зроби так, щоб для заміни шини потрібно було натиснути B!
