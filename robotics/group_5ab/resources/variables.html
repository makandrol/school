📊 Змінні та логіка

Змінні, умови, цикли та функції у MakeCode

Змінні

Змінна — це «коробка» з назвою, в якій зберігається число або текст.

задати x значення 0 — створити змінну та присвоїти значення

змінити x на 1 — додати число до змінної (можна від'ємне!)

x — використати значення змінної

Умови (якщо / інакше)

якщо ... тоді — виконати блоки тільки якщо умова правдива

якщо ... тоді ... інакше — два варіанти дій

і / або — поєднати умови

Приклад: якщо x = паливоX і y = паливоY

Порівняння

x = 5 — дорівнює

x > 4 — більше за

x < 0 — менше за

x ≠ 5 — не дорівнює

Математика

випадкове число від 0 до 4 — генерує випадкове число

залишок від ділення — корисно для циклічних лічильників

Цикли

завжди — повторювати безкінечно (основний цикл гри)

повторити 4 рази — повторити фіксовану кількість разів

поки ... виконувати — повторювати поки умова правдива

Функції

Функція — іменована частина програми для повторного використання.

функція назваФункції — створити функцію (блоки всередині)

викликати назваФункції — виконати функцію
