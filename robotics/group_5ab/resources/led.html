🖥️ LED-дисплей 5×5

Основи роботи з LED-матрицею micro:bit

Координати

Дисплей має 25 діодів у сітці 5×5. Координати починаються з (0,0) у верхньому лівому куті.

x — стовпець (0–4, зліва направо)   y — рядок (0–4, зверху вниз)

0,01,02,03,04,0 0,11,12,13,14,1 0,21,22,23,24,2 0,31,32,33,34,3 0,41,42,43,44,4

Центральний діод: x=2, y=2

Основні блоки

показати значок — вивести іконку (серце, смайлик тощо)

показати рядок "текст" — прокрутити текст на екрані

показати число 5 — показати число

показати світлодіоди — намалювати довільний паттерн 5×5

очистити екран — вимкнути всі діоди

Точкове керування

засвітити x y — увімкнути один діод за координатами

погасити x y — вимкнути один діод

Приклад: засвітити x 2 y 4 — засвітить діод у 3-му стовпці, 5-му рядку

Перевірка кордонів

Координати мають бути в межах 0–4. Якщо вийти за межі, діод не буде видно!

Завжди перевіряй: якщо x > 4 → задати x = 4, якщо x < 0 → задати x = 0
