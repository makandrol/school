### 📌 Урок 23. 🚗 Машинка на micro:bit — «Машинка», «Збирач палива» та «Не зачепи!»
**Дата:** 17.03.2026

#### 🎯 Що ми вивчили сьогодні:
- **Перевірка кордонів** — як зробити, щоб діод не виходив за межі LED-матриці (0 ≤ x ≤ 4)
- Керування рухом діода кнопками A, B, A+B
- Перевірка зіткнень — чи два об'єкти на одному місці (блок `and`)
- Змінна-прапорець (`гра`) — для зупинки програми при Game Over

#### 💻 Що ми зробили:

**⭐ Завдання 1 — Машинка 🚗**
Машинка (діод) їде по нижньому рядку (y=4). Керуємо кнопками:

| Кнопка | Дія |
|--------|-----|
| **A** | Повернути вправо → (x + 1) |
| **B** | Повернути вліво ← (x − 1) |

Головне — **перевірка кордонів**: якщо x > 4, ставимо x = 4; якщо x < 0, ставимо x = 0.

Змінна: `x` (починаємо з 2)

Блоки (on start): `set x to 2`
Блоки (forever): `clear screen` → `plot x (x) y 4` → `pause 100`
Блоки (on button A): `change x by 1` → `if x > 4` → `set x to 4`
Блоки (on button B): `change x by −1` → `if x < 0` → `set x to 0`

**⭐⭐ Завдання 2 — Збирач палива ⛽**
Машинка їздить по всьому полю 5×5! На екрані з'являється «паливо» (випадковий діод) — збери його!

| Кнопка | Дія |
|--------|-----|
| **A** | Рух вправо → |
| **B** | Рух вліво ← |
| **A+B** | Рух вниз ↓ |

Змінні: `x`, `y`, `паливоX`, `паливоY`, `бали`

- Машинка починає в центрі (2,2)
- Паливо — випадковий діод
- Коли машинка збігається з паливом → `бали + 1`, нове паливо
- Зібрав 5 бочок → показує "WIN"!
- Кордони перевіряємо для x **і** y

Для перевірки двох умов одночасно використовуй блок `and` (категорія **Logic**).

**⭐⭐⭐ Завдання 3 — Не зачепи! 💥**
Перешкоди летять зверху вниз! Керуй машинкою і не врізайся!

Змінні: `x`, `перешкодаX`, `перешкодаY`, `бали`, `гра`

- Машинка на нижньому рядку (y=4), керування як у Завданні 1
- Перешкода починає з y=0, рухається вниз
- Якщо перешкода пролетіла вниз → `бали + 1`, нова перешкода
- Якщо перешкода торкнулась машинки → Game Over! Показує "X" та кількість балів
- Змінна `гра` = 1 (грає) / 0 (кінець) — щоб зупинити forever
