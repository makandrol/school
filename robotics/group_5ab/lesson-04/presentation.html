🚗 Машинка на micro:bit

17.03.2026 | Група: 5-А, 5-Б

«Машинка» • «Збирач палива» • «Не зачепи!»

🔄 Пам'ятаємо координати LED

Минулого разу ми вивчили plot x y та unplot x y.

Сьогодні ми будемо керувати машинкою — діодом, який рухається по LED-матриці!

  • x — стовпець (0-4, зліва → направо)
  • y — рядок (0-4, зверху → вниз)
💡 Сьогодні ключове: перевірка кордонів — щоб машинка не «вилетіла» за межі екрану!

x →

0,01,02,03,04,0 0,11,12,13,14,1 0,21,22,23,24,2 0,31,32,33,34,3 0,41,42,43,44,4

y ↓

🆕 Перевірка кордонів

Коли машинка рухається кнопками, вона може «вилетіти» за край екрану!

ПроблемаРішення
x стає менше 0if x < 0 → set x to 0
x стає більше 4if x > 4 → set x to 4
y стає менше 0if y < 0 → set y to 0
y стає більше 4if y > 4 → set y to 4
💡 Це як стіни кімнати — машинка впирається і далі не їде!

⭐ Завдання 1 — Машинка 🚗

Машинка їде по нижньому рядку (y = 4). Керуй кнопками!

КнопкаДія
AПовернути вправо → (x + 1)
BПовернути вліво ← (x − 1)
  • Машинка — це один діод
  • Рухається тільки вліво-вправо
  • Не може вилізти за кордони! (0 ≤ x ≤ 4)

Машинка на рядку y=4:

🚗

B: ← вліво   |   A: вправо →

⚠️ Запам'ятай: A — вправо (+1), B — вліво (−1)!

🛠️ Будуємо «Машинку» — блоки

Створи змінну: x

on start:

set x to 2

forever:

clear screen

plot x x y 4

pause 100

on button A pressed:

change x by 1

if x > 4

set x to 4

on button B pressed:

change x by −1

if x < 0

set x to 0

💡 Без перевірки кордонів машинка «зникне» за межі екрану і ти її не побачиш!

⭐⭐ Завдання 2 — Збирач палива ⛽

Машинка їздить по всьому полю 5×5! На екрані з'являється «паливо» — збери його!

КнопкаДія
AРух вправо → (x + 1)
BРух вліво ← (x − 1)
A+BРух вниз ↓ (y + 1)
  • Машинка їде в 3 напрямках
  • Паливо — випадковий діод на екрані
  • Зібрав 5 бочок = перемога! 🏆
  • Кордони перевіряємо для x і y!

Машинка + паливо:

⛽ 🚗

Машинка (1,2) збирає паливо (3,0)

🏆 Хто збере 5 бочок найшвидше?

🛠️ Будуємо «Збирач палива» — блоки

Змінні: x, y, паливоX, паливоY, бали

on start:

set x to 2

set y to 2

set бали to 0

set паливоX to pick random 0 to 4

set паливоY to pick random 0 to 4

forever:

clear screen

plot x x y y

plot x паливоX y паливоY

if x = паливоX and y = паливоY

change бали by 1

set паливоX to pick random 0 to 4

set паливоY to pick random 0 to 4

if бали = 5

show string "WIN"

pause 200

on button A (вправо):

change x by 1

if x > 4 → set x to 4

on button B (вліво):

change x by −1

if x < 0 → set x to 0

on button A+B (вниз):

change y by 1

if y > 4 → set y to 4

⚠️ Для перевірки двох умов одночасно (x = паливоX і y = паливоY), використай блок and з категорії Logic!

⭐⭐⭐ Завдання 3 — Не зачепи! 💥

Перешкоди летять зверху вниз! Керуй машинкою і не врізайся!

ЕлементОпис
🚗 МашинкаНижній рядок (y=4), рухається вліво/вправо
💥 ПерешкодаПочинає з y=0, рухається вниз
🏆 Бали+1 за кожну перешкоду, яку оминув
  • Перешкода — діод у випадковому стовпці
  • Рухається зверху вниз
  • Якщо торкнулась машинки — Game Over!

Перешкода летить:

💥 🚗

Не зачепи! A → вправо, B → вліво

💡 Це як справжня гра на телефоні — тільки на micro:bit!

🛠️ Будуємо «Не зачепи!» — блоки

Змінні: x, перешкодаX, перешкодаY, бали, гра

on start:

set x to 2

set бали to 0

set гра to 1

set перешкодаX to pick random 0 to 4

set перешкодаY to 0

forever:

if гра = 1

clear screen

plot x x y 4

plot x перешкодаX y перешкодаY

change перешкодаY by 1

forever (продовження):

if перешкодаY > 4

set перешкодаY to 0

set перешкодаX to pick random 0 to 4

change бали by 1

if перешкодаY = 4 and перешкодаX = x

set гра to 0

show string "X"

show number бали

pause 400

Кнопки A / B:

Такі самі як у Завданні 1!

  • ✅ Зіткнення — перевірка, чи два об'єкти на одному місці
  • ✅ Змінна-прапорець (гра) — для зупинки програми
