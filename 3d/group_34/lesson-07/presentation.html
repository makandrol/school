⭐ Завдання 1 — Рибка з водоростями 🐟🌿

🐟 Рибка:

  1. Sphere → розтягни в овал: 40×25×20, помаранчевий
  2. Cone (хвіст) — ⌀20, H=15 → поверни на бік → ззаду
  3. Sphere (око) — ⌀6 білий + ⌀2 чорний (зіниця)
  4. Box (плавець зверху) — 15×8×2, жовтий

🌿 Водорості на камені:

  1. Half Sphere (камінь) — ⌀20, сірий
  2. Cylinder (стебло) — ⌀3, H=25, зелений ×3 → нахили кожне
  3. Sphere (листок) — ⌀8, зелений → розтягни
💡 Нахили водорості в різні боки — буде як справжні!

⭐⭐ Завдання 2 — Акула з зубами 🦈

🦈 Акула:

  1. Sphere → довгий овал 60×20×18, сірий
  2. Cone (морда) — ⌀18, H=20, сірий → спереду
  3. Box (хвіст) — 5×20×15, під кутом ззаду
  4. Box (спинний плавець) — 3×15×12, зверху
  5. Cone (зуб!) — ⌀2, H=4, білий → Ctrl+D ×6-8 в ряд
  6. Sphere (очі) — ⌀4 білий + ⌀2 чорний ×2
  7. Box (бокові плавці) — 3×12×8, ×2

🐟 Маленька рибка:

  1. Sphere овал 15×10×8, жовтий + Cone хвостик
🦈 Зуби — це маленькі конуси! Зроби один → Ctrl+D → розстав у ряд!

⭐⭐⭐ Завдання 3 — Підводний човен і корали 🚢🪸

🚢 Човен (Yellow Submarine!):

  1. Cylinder (корпус) — ⌀25, H=60, жовтий → на бік
  2. Half Sphere — ⌀25, ніс + хвіст
  3. Cylinder (рубка) — ⌀15, H=15, зверху
  4. Sphere → Hole (ілюмінатор) ⌀6, ×3 → Group
  5. Cylinder (перископ) — ⌀3, H=20
  6. Box (кермо) — 2×15×10, ×2 хрестом

🪸 Кораловий риф:

  1. Half Sphere (дно) — ⌀30, пісочний
  2. Cylinder (корал) — ⌀6, H=20, червоний ×3 різної висоти
  3. Sphere (бульбашки) — ⌀4, рожевий
  4. Cylinder (водорість) — ⌀3, H=15, зелений ×2
🎨 Жовтий човен як Yellow Submarine! Корали — яскраві: червоний, рожевий, помаранчевий!

⭐⭐⭐⭐ Завдання 4 — Восьминіг зі скарбами 🐙💰

🐙 Восьминіг:

  1. Sphere (голова) — ⌀35, фіолетовий
  2. Sphere (очі) — ⌀8 білий + ⌀3 чорний ×2
  3. Cylinder (щупальце) — ⌀5, H=40 → нахили ~20°
  4. Ctrl+D → поверни на 45° → Ctrl+D ×6 = 8 щупалець!
  5. Sphere (присоски) — ⌀2, рожевий (бонус)

💰 Скриня зі скарбами:

  1. Box (скриня) — 30×20×18, коричневий
  2. Box (кришка) — 30×20×3, нахили (відчинена!)
  3. Cylinder (монети) — ⌀4, H=1, золотий ×5-6
  4. Sphere (перлина) — ⌀5, білий
  5. Half Sphere (мушля) — ⌀8, рожевий
🏆 Найскладніше! 360° ÷ 8 = 45° для кожного Ctrl+D. Покажи вчителю!
