# Заняття 7: 🐠 Підводний світ
**Дата:** 31.03.2026
**Статус:** ✅ Створено

## Тема
Підводний світ: моделювання морських тварин та підводного транспорту. Від простої рибки до складного восьминіга з 8 щупальцями. Закріплення Ctrl+D з точним обертанням.

## Нові концепції
- **Створення овалу**: розтягування Sphere в одному напрямку
- **Дублювання з точним кутом**: 360° ÷ N = кут повороту (72° для 5, 45° для 8)
- **Автоматичне повторення повороту**: після першого Ctrl+D + повороту, наступні Ctrl+D зберігають кут

## Що зробили
- ⭐ **Рибка** — Sphere розтягнута в овал (40×25×20, помаранчевий) + Cone (хвіст, повернутий на бік) + Sphere (око, ⌀5 білий + ⌀2 чорний) + Box (плавець, тонкий, зверху)
- ⭐⭐ **Морська зірка** — Cylinder (⌀8, H=30) + Cone (кінчик) → Group → Ctrl+D × 4 з поворотом на 72° + Sphere (центр, ⌀15) → Align
- ⭐⭐⭐ **Підводний човен** — Cylinder (корпус, ⌀25, H=60, на боці) + Half Sphere (ніс/хвіст) + Cylinder (вежа, ⌀15) + Sphere Hole (ілюмінатори, ×3) + Cylinder (перископ, ⌀3) + Box (кермо хвоста)
- ⭐⭐⭐⭐ **Восьминіг** — Sphere (голова, ⌀35, фіолетовий) + Sphere (очі з зіницями) + Cylinder (щупальце, ⌀5, H=40, повернуте) → Ctrl+D × 7 з поворотом на 45°

## Закріплені навички
- Ctrl+D з обертанням на точний кут (72°, 45°)
- Hole + Group для ілюмінаторів
- Align для центрування
- Створення овалу зі Sphere
- Складання моделей з багатьох деталей
