# Заняття 7: 🐠 Підводний світ
**Дата:** 31.03.2026
**Статус:** ✅ Створено

## Тема
Підводний світ: моделювання морських сцен з кількома об'єктами. Кожне завдання — ціла композиція з 2+ моделей. Від рибки з водоростями до восьминіга зі скринею скарбів.

## Нові концепції
- **Створення овалу**: розтягування Sphere в одному напрямку
- **Конуси як деталі**: маленькі Cone для зубів акули
- **Дублювання з точним кутом**: 360° ÷ 8 = 45° для щупалець восьминіга
- **Композиція з кількох об'єктів**: кожне завдання — сцена, а не окремий предмет

## Що зробили
- ⭐ **Рибка з водоростями** — Sphere-овал (тіло) + Cone (хвіст) + око + плавці + Half Sphere (камінь) + Cylinder (стебла водоростей ×3, нахилені) + Sphere (листки)
- ⭐⭐ **Акула з зубами** — довгий Sphere-овал (60×20×18, тіло) + Cone (морда) + Box (хвіст, плавці) + Cone ×6-8 (зуби! Ctrl+D) + маленька рибка-жертва (Sphere + Cone)
- ⭐⭐⭐ **Підводний човен і корали** — Cylinder (корпус жовтий на боці) + Half Sphere (ніс/хвіст) + Cylinder (рубка) + Sphere-Hole (ілюмінатори ×3) + перископ + кораловий риф (Cylinder різної висоти + Sphere бульбашки + водорості)
- ⭐⭐⭐⭐ **Восьминіг зі скринею скарбів** — Sphere (голова) + очі + Cylinder-щупальця ×8 (Ctrl+D на 45°) + Box (скриня) + Box (нахилена кришка) + Cylinder (монети) + Sphere (перлина) + Half Sphere (мушля)

## Закріплені навички
- Ctrl+D з обертанням на точний кут (45°)
- Hole + Group для ілюмінаторів
- Align для центрування
- Створення овалу зі Sphere
- Cone як деталь (зуби акули)
- Композиція з кількох моделей (сцена)

## Роздатковий матеріал
- Не використовується (лише презентація)
