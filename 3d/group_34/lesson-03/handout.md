# 🎨 3D-моделювання | Заняття 3
**Дата:** 24.02.2026 | **Тема:** Будиночок. Обертання та вирівнювання

### Нові інструменти:
- **Обертання:** клікни на об'єкт → тягни за вигнуту стрілку (дугу). Shift = по 45°
- **Align (вирівнювання):** виділи 2 об'єкти → натисни **L** → клікни на центральну точку

# ⭐ Завдання 1 (основне): «Будиночок»

**Що моделюємо:** простий будиночок з дахом, дверима, вікнами та трубою

### Покрокова інструкція:

**Крок 1.** Стіни
- Перетягни **Box** (куб) на площину
- Зміни розмір: **ширина 60, глибина 50, висота 40**
- Це стіни нашого будиночка

**Крок 2.** Дах
- Перетягни ще один **Box** на площину
- Зміни розмір: **ширина 65, глибина 55, висота 40**
- Тепер **оберни** його: клікни → побачиш дуги → поверни на **45°** (тримай Shift!)
- Підніми дах **вгору** — постав його над стінами
- Щоб дах був точно по центру: виділи стіни + дах (Shift + клік) → натисни **L** → клікни на центральну точку по X та по Y

**Крок 3.** Двері
- Перетягни **Box** на площину
- Зміни розмір: **ширина 12, глибина 5, висота 25**
- Зміни його на **Hole** (натисни → вибери Hole)
- Постав його на передню стінку будиночка, **внизу по центру**
- Він має «входити» в стіну

**Крок 4.** Вікна
- Перетягни **Box** → розмір: **ширина 10, глибина 5, висота 10**
- Зміни на **Hole**
- Постав на передню стінку, **зліва від дверей**, на середній висоті
- **Ctrl + D** (дублюй) → перетягни копію **праворуч від дверей**
- Тепер у тебе два вікна!

**Крок 5.** Труба
- Перетягни **Box** → розмір: **ширина 8, глибина 8, висота 20**
- Постав його **зверху на дах**, трохи збоку
- Підніми, щоб він стирчав над дахом

**Крок 6.** Групуємо
- Виділи всі об'єкти: **Ctrl + A**
- Натисни **Ctrl + G** (Group)
- Двері та вікна «вирізались» зі стін!

✅ **Готово!** Ти змоделював(ла) будиночок! 🏠

**Бонус:** зміни кольори — стіни жовті, дах червоний, труба коричнева!

# ⭐⭐ Завдання 2 (додаткове): «Дерево»

**Що моделюємо:** дерево з товстим стовбуром та пишною кроною

### Покрокова інструкція:

**Крок 1.** Стовбур
- Перетягни **Cylinder** (циліндр) на площину
- Розмір: **ширина 10, глибина 10, висота 40**
- Колір: коричневий

**Крок 2.** Крона (нижній ярус)
- Перетягни **Sphere** (сфера) на площину
- Розмір: **ширина 45, глибина 45, висота 35**
- Підніми її вгору — постав на верхівку стовбура
- Колір: зелений

**Крок 3.** Крона (верхній ярус)
- **Ctrl + D** — дублюй сферу
- Зменши копію: **ширина 35, глибина 35, висота 30**
- Підніми ще вище — постав зверху на першу сферу
- Вирівняй: виділи обидві сфери → **L** → по центру

**Крок 4.** Групуємо
- **Ctrl + A** → **Ctrl + G**

✅ **Готово!** Ти змоделював(ла) дерево! 🌳

**Бонус:** постав дерево поруч із будиночком!

# ⭐⭐⭐ Завдання 3 (для швидких): «Сніговик»

**Що моделюємо:** сніговика з трьох куль, з носом-морквою, очима та капелюхом

### Підказки:

**Тіло:**
- Три **Sphere** (сфери), одна на одній
- Нижня — найбільша (40×40×40), середня менша (30×30×30), верхня — найменша (22×22×22)
- Вирівняй їх по центру (**L**)!

**Обличчя:**
- Очі: дві маленькі **Sphere** (чорні, розмір 3×3×3)
- Ніс-морква: **Cone** (конус), оранжевий, повернутий **горизонтально** (обертання на 90°!)
- Рот: кілька крихітних сфер (чорних) дугою

**Капелюх:**
- Поля: **Cylinder** — плоский (30×30×3), чорний
- Тулія: **Cylinder** (15×15×15), чорний, поставити зверху на поля

**Руки:**
- Два **Cylinder** — тонкі та довгі, повернути під кутом

- **Ctrl + A** → **Ctrl + G**

✅ **Готово!** ⛄

# ⭐⭐⭐⭐ Завдання 4 (творче): «Вулиця»

**Що моделюємо:** цілу вулицю з кількох будиночків, дерев та інших об'єктів!

### Що потрібно зробити:

Створи сцену, в якій є **щонайменше**:
- 🏠 **2-3 будиночки** (можна різних розмірів і кольорів)
- 🌳 **1-2 дерева**
- 🛤️ **Дорога** — плоский довгий Box (сірий колір)

### Додаткові ідеї (на твій вибір):
- 🚗 Машина (кузов = Box + колеса = Cylinder, повернутий на 90°)
- 🏫 Школа (більший будиночок з багатьма вікнами)
- 🌻 Клумба з квітами (маленькі кольорові циліндри + сфери)
- 🪑 Лавочка (з кількох Box)
- ☀️ Сонечко (Sphere жовтого кольору, піднята високо)

### Правила:
- Мінімум **8 об'єктів** в сцені
- Використай **хоча б 3 різні кольори**
- Використай **обертання** хоча б 1 раз
- Використай **Align** хоча б 1 раз

**Покажи вчителю коли закінчиш — найкращі роботи покажемо всьому класу!** 🏆
