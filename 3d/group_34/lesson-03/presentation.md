# 3D-моделювання | Заняття 3

## 🏠 Будиночок. Обертання та вирівнювання

**24.02.2026 | Група: 3-4 клас**

---

## 🆕 Обертання об'єктів

Іноді фігуру потрібно **повернути** — наприклад, зробити дах!

**Як обертати:**
1. Клікни на об'єкт
2. Побачиш **три вигнуті стрілки** (дуги) 🔄
3. Потягни за дугу — об'єкт повертається!

**Підказка:** тримай **Shift** — повертає рівно по **45°**

> Спробуй повернути куб на 45° — він стане схожий на діамант! 💎

---

## 🆕 Align — Вирівнювання

Коли потрібно поставити один об'єкт **точно по центру** іншого:

1. Виділи **два або більше** об'єктів (тримай **Shift** + клік)
2. Натисни кнопку **Align** (або клавішу **L**)
3. З'являться **чорні точки** — клікни на центральну
4. Об'єкти стануть рівно один відносно іншого!

> Це дуже зручно для даху — він стане точно по центру стіни!

---

## 🏠 Завдання на сьогодні

Будемо моделювати **будиночок** з:
- **Стіни** → Box
- **Дах** → Box, повернутий на 45°
- **Двері** → Box (Hole)
- **Вікна** → Box (Hole)
- **Труба** → Box

### Ще завдання (для тих, хто швидко):
- 🌳 **Дерево** — стовбур + крона
- ⛄ **Сніговик** — три сфери
- 🏘️ **Вулиця** — кілька будиночків разом!

> Детальна інструкція — на вашому аркуші! 📄
