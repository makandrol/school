# Заняття 3: 🏠 Будиночок. Обертання та вирівнювання
**Дата:** 24.02.2026
**Статус:** 📅 Запланований

## Тема
Обертання об'єктів та вирівнювання (Align). Проєкт «Будиночок».

## Нові концепції
- **Обертання:** клік на об'єкт → тягни за дугу-стрілку. Shift = кроки по 45°
- **Align (вирівнювання):** виділити 2+ об'єкти → кнопка Align або клавіша L → клік на центральну точку
- **Duplicate (Ctrl+D):** дублювання об'єктів (для вікон, дерев тощо)

## Що зробили
- ⭐ **Будиночок** — стіни (Box 60×50×40), дах (Box 65×55×40 повернутий на 45°), двері та вікна (Box → Hole), труба
- ⭐⭐ **Дерево** — стовбур (Cylinder) + крона з 2 сфер, вирівняних по центру
- ⭐⭐⭐ **Сніговик** — три сфери + ніс-морква (Cone повернутий на 90°) + капелюх
- ⭐⭐⭐⭐ **Вулиця** — творче завдання: кілька будиночків + дерева + дорога

## Закріплені навички
- Додавання фігур, переміщення, масштабування
- Hole + Group для вирізання
- НОВЕ: Обертання на конкретні кути
- НОВЕ: Align для точного позиціонування
- НОВЕ: Duplicate (Ctrl+D) для швидкого копіювання
