🏠 Будиночок. Обертання та вирівнювання

24.02.2026 | Група: 3-4 клас

🆕 Обертання об'єктів

Іноді фігуру потрібно повернути — наприклад, зробити дах!

Як обертати:

  • Клікни на об'єкт
  • Побачиш три вигнуті стрілки (дуги) 🔄
  • Потягни за дугу — об'єкт повертається!
💡 Тримай Shift — повертає рівно по 45°. Спробуй повернути куб на 45° — він стане схожий на діамант! 💎

🆕 Align — Вирівнювання

Коли потрібно поставити один об'єкт точно по центру іншого:

  • Виділи два або більше об'єктів (тримай Shift + клік)
  • Натисни кнопку Align (або клавішу L)
  • З'являться чорні точки — клікни на центральну
  • Об'єкти стануть рівно один відносно іншого!
Це дуже зручно для даху — він стане точно по центру стіни!

🏠 Завдання на сьогодні

Будемо моделювати будиночок:

  • Стіни → Box
  • Дах → Box, повернутий на 45°
  • Двері → Box (Hole)
  • Вікна → Box (Hole)
  • Труба → Box

Ще завдання (для тих, хто швидко):

  • 🌳 Дерево — стовбур + крона
  • ⛄ Сніговик — три сфери
  • 🏘️ Вулиця — кілька будиночків разом!
📄 Детальна інструкція — на вашому аркуші!
