⭐ Завдання 1: «Ракета» 🚀

Космічна ракета з корпусом, носовим конусом, стабілізаторами та вікном!

Ракета: корпус + конус + стабілізатори + вікно
Крок 1. Корпус — Cylinder → діаметр 20, висота 50 → колір білий
Крок 2. Носовий конус — Cone (Basic Shapes) → діаметр 20, висота 20 → колір червоний
Крок 3. Підняти конус — тягни чорний конус ↑ → підніми на 50 мм
Крок 4. Вирівняти — виділи корпус + конус → L (Align) → клікни центральні точки
Крок 5. Стабілізатор — Box → 3 × 10 × 15 → колір червоний → постав збоку внизу корпусу
Крок 6. Ще стабілізатори — Ctrl+D → поверни на 120° → Ctrl+D → ще на 120° (всього 3 штуки)
Крок 7. Вікно (ілюмінатор) — Sphere → діаметр 10 → колір блакитний → постав на середину корпусу
Крок 8. Групуємо — Ctrl+A → Ctrl+G
✅ Готово! Космічна ракета! 🚀

🎨 Бонус: додай сопло (Cylinder, помаранчевий, внизу) або напис на корпусі (Text)

⭐⭐ Завдання 2: «Планета Сатурн» 🪐

Планета з кільцем і маленькими місяцями!

Сатурн: сфера + кільце (Cylinder − Hole) + місяці
Крок 1. Планета — Sphere → діаметр 40 → колір жовто-золотий (#F1C40F)
Крок 2. Кільце (зовнішнє) — Cylinder → діаметр 70, висота 2 → колір помаранчевий (#E67E22)
Крок 3. Отвір у кільці — Cylinder → діаметр 42, висота 3 → зроби Hole
Крок 4. Вирізаємо — Align кільце + Hole по центру → виділи обидва → Ctrl+G
Крок 5. Збираємо — підніми кільце на 19 мм (центр планети) → Align планету + кільце
Крок 6. Місяць 1 — Sphere → діаметр 8 → колір сірий → постав збоку, підніми трохи
Крок 7. Місяць 2 — Ctrl+D → зменши розмір → постав з іншого боку
Крок 8. Групуємо — Ctrl+A → Ctrl+G
✅ Готово! Планета Сатурн з кільцем! 🪐

🎨 Бонус: нахили кільце трохи (обертання по X або Z) для реалістичності!

💡 Хитрість з кільцем: Hole-циліндр має бути трохи більшим за планету (⌀42 > ⌀40), щоб кільце не перетиналось з планетою.

⭐⭐⭐ Завдання 3: «Космонавт» 👨‍🚀

Космонавт у скафандрі із золотим забралом і рюкзаком!

Космонавт: шолом + тіло + руки + ноги + рюкзак
Крок 1. Шолом — Sphere → діаметр 20 → колір білий
Крок 2. Тіло — Box → 18 × 14 × 25, Radius 3 → колір білий → підняти так, щоб голова була зверху
Крок 3. Рука ліва — Cylinder → діаметр 6, висота 22 → колір білий → поставити збоку тіла зліва
Крок 4. Рука права — Ctrl+D → перетягнути на правий бік
Крок 5. Нога ліва — Cylinder → діаметр 7, висота 20 → колір білий → поставити внизу тіла зліва
Крок 6. Нога права — Ctrl+D → перетягнути вправо
Крок 7. Рюкзак — Box → 14 × 8 × 20 → колір сірий → поставити ззаду тіла
Крок 8. Забрало — Half Sphere (або Sphere) → діаметр 14 → колір золотий (#F39C12) → поставити на шолом спереду
Крок 9. Групуємо — Ctrl+A → Ctrl+G
✅ Готово! Космонавт у скафандрі! 👨‍🚀

🎨 Бонус: додай прапорець України (Box, синій + жовтий) на руку або деталі на рюкзаку!

⭐⭐⭐⭐ Завдання 4: «Космічна станція» 🛸

Побудуй власну космічну станцію з модулів, сонячних панелей та антени!

Космічна станція: модулі + сонячні панелі + антена

Що потрібно зробити:

  • Головний модуль — Cylinder → ⌀15, H=50 → колір білий → поверни горизонтально (90°)
  • Бічні модулі — Cylinder → менші → приєднай з боків
  • З'єднання — Cylinder → тонкий (⌀3) → між модулями
  • Сонячні панелі — Box → 30 × 20 × 0.5 → колір синій → по боках на тонких стрижнях
  • Антена — Cylinder → ⌀2, H=15 → зверху + Sphere або Cone на кінці

Додаткові ідеї:

  • 🔴 Додай червоні вогні на кінці сонячних панелей
  • 🏷️ Напиши назву станції (Text) на головному модулі
  • 🇺🇦 Зроби українську станцію з синьо-жовтими кольорами!
  • 🛸 Додай маленький космічний корабель поруч
💡 Підказка: щоб покласти циліндр горизонтально, поверни його на 90° по осі X або Y (дуга-стрілка при виділенні).
🏆 Покажи вчителю — найкращі роботи покажемо всьому класу!
