# Заняття 4: 🎨 Кольори та 3D-текст. Іменна табличка
**Дата:** 03.03.2026
**Статус:** 📅 Запланований

## Тема
Кольори об'єктів, 3D-текст (Text shape), піднімання об'єктів над площиною. Проєкт «Іменна табличка».

## Нові концепції
- **Кольори:** клік на об'єкт → панель кольору справа → вибрати колір або HEX-код
- **3D-текст (Text):** Basic Shapes → Text → перетягнути → у Inspector змінити текст і висоту
- **Піднімання об'єктів:** чорний конус ↑ зверху об'єкта → тягнути вгору (для розміщення тексту на основі)

## Що зробили
- ⭐ **Іменна табличка** — Box 80×30×5 (кольорова основа) + Text (ім'я, висота 8) зверху, Align по центру
- ⭐⭐ **Медаль** — Cylinder (диск) + зірка/текст + стрічка (Box)
- ⭐⭐⭐ **Брелок** — форма + отвір для кільця (Cylinder → Hole) + ініціали (Text)
- ⭐⭐⭐⭐ **Вивіска магазину** — творче: назва магазину + символ + мінімум 3 кольори

## Закріплені навички
- Hole + Group для вирізання (повторення)
- Align для позиціонування (повторення)
- Duplicate / Ctrl+D (повторення)
- НОВЕ: Зміна кольорів об'єктів
- НОВЕ: Додавання 3D-тексту
- НОВЕ: Піднімання об'єктів (чорний конус)
