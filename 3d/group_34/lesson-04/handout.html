Тема: Кольори та 3D-текст. Іменна табличка

🆕 Нові інструменти

🎨 КольориКлікни на об'єкт → справа з'явиться палітра → вибери колір
🔤 3D-текст (Text)На лівій панелі: Basic Shapes → прокрути вниз → Text → перетягни
⬆️ Підняти об'єктКлікни на об'єкт → тягни за чорний конус ↑ зверху

🎨 Як змінити колір?

1. Клікни на фігуру → 2. Справа натисни на кружечок кольору → 3. Вибери!

💡 Кожна фігура може мати свій колір. Колір зберігається навіть після групування.

🔤 Як додати текст?

1. Зліва: Basic Shapes → прокрути вниз → знайди "Text"

2. Перетягни на площину → клікни → справа введи своє ім'я

3. Зміни висоту (Height) на 8–10

⭐ Завдання 1: «Іменна табличка»

Створи табличку зі своїм ім'ям — основа + 3D-текст + кольори!

Іменна табличка — основа + текст з ім'ям
Крок 1. Основа — перетягни Box → зміни розміри: 80 × 30 × 5
Крок 2. Колір основи — клікни на Box → вибери колір (синій, зелений — будь-який!)
Крок 3. Додай текст — ліва панель → Text → перетягни → клікни → справа введи своє ім'я
Крок 4. Налаштуй текст — висота (Height): 8. Вибери колір тексту (жовтий або білий добре видно)
Крок 5. Вирівняй — виділи і текст і основу → натисни L (Align) → клікни на центральні точки
Крок 6. Підніми текст — клікни на текст → тягни за чорний конус ↑ зверху → підніми на 5 мм
Крок 7. Згрупуй — Ctrl+A → Ctrl+G
✅ Готово! Іменна табличка створена! 🏷️

🎨 Бонус: додай рамку навколо (тонкий Box → Hole по центру) або зроби кути заокругленими

⭐⭐ Завдання 2: «Медаль»

Круглий диск + текст або зірка зверху + стрічка

Медаль — Cylinder + зірка або текст
Крок 1. Диск — Cylinder (циліндр): діаметр 40, висота 3 → колір золотий (#F1C40F)
Крок 2. Зірка або текст — Text з "⭐" або "#1" → висота 5 → підняти на 3 мм → Align по центру диска
Крок 3. Стрічка — Box 8×20×2 → колір червоний → поставити зверху диска → підняти
Крок 4. Групуємо — Ctrl+A → Ctrl+G
✅ Готово! Медаль створена! 🏅

🎨 Бонус: додай ще одне кільце навколо (Cylinder трохи більший → Hole по центру) або напиши текст "ПЕРЕМОЖЕЦЬ"

⭐⭐⭐ Завдання 3: «Брелок»

Плоска форма + отвір для кільця + ініціали або символ

Брелок — форма + отвір + текст
Крок 1. Основа — Box 30×40×5 → заокругли (Radius = 4) → обери колір
Крок 2. Отвір для кільця — Cylinder діаметр 6, висота 10 → зроби Hole → постав у верхній частині
Крок 3. Вирізаємо — виділи основу + отвір → Ctrl+G (групуємо — отвір вирізається!)
Крок 4. Текст — Text → вписати ініціали (перші літери імені та прізвища) → висота 6 → Align → підняти на 5 мм
Крок 5. Групуємо — Ctrl+A → Ctrl+G
✅ Готово! Брелок створено! 🔑

🎨 Бонус: замість ініціалів — зроби символ (серце, зірка, тваринка)

⭐⭐⭐⭐ Завдання 4: «Вивіска магазину»

Придумай свій магазин і зроби для нього вивіску!

Приклад: вивіска кафе-морозива

Що потрібно зробити:

  • Основа — великий Box (мінімум 100×40×5)
  • Назва магазину — Text зверху
  • Символ — що продає магазин (фігура з Basic Shapes або щось творче)
  • Мінімум 3 різні кольори

Ідеї для магазинів:

  • 🍕 Піцерія — кружечок + трикутники
  • 🎮 Ігровий магазин — кубик або джойстик
  • 📚 Книгарня — прямокутники-книжки
  • 🐾 Зоомагазин — силует тваринки
  • 🚀 Космічний магазин — ракета або зірка
🏆 Покажи вчителю — найкращі роботи покажемо всьому класу!
