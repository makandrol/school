### Нові інструменти:

**Кольори:**
1. Клікнути на об'єкт (або групу)
2. Праворуч з'явиться палітра кольорів
3. Вибрати колір кліком АБО ввести HEX-код (#FF0000 = червоний)
4. Кожна фігура може мати свій колір

**3D-текст (Text):**
1. На лівій панелі: Basic Shapes → прокрутити вниз → знайти **"Text"**
2. Перетягнути на робочу площину
3. Клікнути на текст → у Inspector (праворуч):
   - **Text:** ввести свій текст (наприклад, ім'я)
   - **Height:** висота літер (рекомендуємо 8-10)
   - **Font:** шрифт (Sans, Serif, Mono)
4. Текст можна рухати, обертати, масштабувати як звичайний об'єкт
5. Текст можна зробити **Hole** — тоді він вирізає літери з іншого об'єкта

---

### Завдання 1 (основне): «Іменна табличка» ⭐
Основа (Box 80×30×5) + текст (ім'я) зверху + колір

**Кроки:**
1. Box 80×30×5 — основа (будь-який колір)
2. Text → вписати своє ім'я → висота 8 → поставити зверху
3. Align текст з основою (по центру)
4. Підняти текст на 5 мм (на основу)
5. Колір тексту — контрастний до основи
6. Ctrl+A → Ctrl+G

### Завдання 2: «Медаль» ⭐⭐
Круглий диск (Cylinder) + зірка або текст + стрічка (тонкий прямокутник)

### Завдання 3: «Брелок» ⭐⭐⭐
Форма на вибір + отвір для кільця (маленький Cylinder → Hole) + ініціали або символ

### Завдання 4 (творче): «Вивіска магазину» ⭐⭐⭐⭐
Прямокутна табличка + назва магазину + символ (наприклад, морозиво для кафе)
