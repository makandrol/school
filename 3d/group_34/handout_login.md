## Крок 3. Ти всередині!

Після входу ти побачиш головну сторінку Tinkercad.

Щоб почати моделювати:
1. Натисни **«Створити новий дизайн»** (або **«Create new design»**)
2. Відкриється робоча площина — можна починати!

## Навігація (нагадування)

| Дія | Як зробити |
|-----|-----------|
| Обертати сцену | Затисни **праву кнопку миші** і рухай |
| Наблизити / віддалити | Крути **колесо миші** |
| Пересунути сцену | Затисни **колесо миші** (середня кнопка) і рухай |
| Додати фігуру | Перетягни з **панелі справа** на площину |
| Виділити об'єкт | **Клік лівою кнопкою** на об'єкті |
| Видалити | Виділи об'єкт → натисни **Delete** |
| Скасувати дію | **Ctrl + Z** |
| Дублювати | **Ctrl + D** |
| Групувати | Виділи все (Ctrl+A) → **Ctrl + G** |
