⭐ Завдання 1: «Стартовий світлофор Ф1» 🚦

На старті Формули-1 — горизонтальна панель з 5 червоними вогнями на стовпі!

Стартовий світлофор Ф1 — стовп + панель + 5 вогнів
Крок 1. Стовп — Cylinder → діаметр 5, висота 60 → колір сірий
Крок 2. Панель — Box → розміри 50 × 12 × 5 → колір чорний
Крок 3. Підняти панель — клікни на панель → тягни чорний конус ↑ → підніми на 55 мм
Крок 4. Вирівняти — виділи стовп + панель → натисни L (Align) → клікни центральні точки
Крок 5. Перший вогонь — Cylinder → діаметр 8, висота 3 → колір червоний → підняти на 59 мм
Крок 6. Поставити зліва — постав кружечок на лівий край панелі
Крок 7. Дублювати! — Ctrl+D → зсунь копію вправо на ~10 мм
Крок 8. Ще 3 вогні — Ctrl+D → Ctrl+D → Ctrl+D (кожен зсувається автоматично!)
Крок 9. Групуємо — Ctrl+A → Ctrl+G
✅ Готово! Стартовий світлофор Ф1! 🚦

🎨 Бонус: додай основу (широкий Box під стовпом) або зроби вогні з маленьким чорним обідком

⭐⭐ Завдання 2: «Траса з розміткою» 🏁

Ділянка гоночної траси з асфальтом, бордюрами (curbs) і стартовою лінією!

Траса — асфальт + бордюри (curbs) + стартова лінія + стартові позиції
Крок 1. Асфальт — Box → 120 × 80 × 2 → колір темно-сірий (#636E72)
Крок 2. Перший бордюр — Box → 10 × 5 × 1 → колір червоний → поставити на верхній край асфальту
Крок 3. Дублювати! — Ctrl+D → зсунути вправо → змінити колір на білий
Крок 4. Повторити — Ctrl+D (червоний) → Ctrl+D (білий) → ... поки не заповниш край
Крок 5. Нижній бордюр — так само, але знизу (почни з білого!)
Крок 6. Стартова лінія — Box → 2 × 60 × 0.5 → колір білий → поставити перпендикулярно
Крок 7. Стартові позиції — Box → 5 × 10 × 0.5 → білий → розставити за лінією (2 в ряд)
✅ Готово! Ділянка гоночної траси! 🏁
🏎️ Бордюри (curbs) — це червоно-білі смуги по краях траси. Пілоти наїжджають на них на поворотах!

⭐⭐⭐ Завдання 3: «Гоночний автомобіль» 🏎️

Спрощений болід Формули-1 — кузов, колеса, спойлер!

Болід Ф1 — кузов + кокпіт + колеса + спойлер
Крок 1. Кузов — Box → 40 × 15 × 6 → Radius 2 → вибери колір команди!

🔴 Ferrari = #E74C3C   🔵 Red Bull = #1A3A6B   🟢 Mercedes = #00D2BE   🟠 McLaren = #FF8000

Крок 2. Кокпіт — Box → 10 × 8 × 5 → зроби Hole → постав зверху кузова ближче до центру
Крок 3. Вирізаємо — виділи кузов + кокпіт-Hole → Ctrl+G (кокпіт вирізається!)
Крок 4. Перше колесо — Cylinder → діаметр 8, висота 4 → колір чорний → поверни на 90° (дуга по осі X)
Крок 5. Поставити колесо — постав збоку кузова (спереду зліва)
Крок 6. Ще 3 колеса — Ctrl+D → постав справа спереду → Ctrl+D → ззаду зліва → Ctrl+D → ззаду справа
Крок 7. Переднє крило — Box → 20 × 3 × 1 → колір кузова → поставити спереду, підняти на 1 мм
Крок 8. Спойлер — Box → 18 × 2 × 8 → колір кузова → поставити ззаду, підняти на 6 мм
Крок 9. Групуємо — Ctrl+A → Ctrl+G
✅ Готово! Болід Ф1! 🏎️

🎨 Бонус: додай номер пілота (Text) на бік кузова або зроби антикрило складнішим

⭐⭐⭐⭐ Завдання 4: «Подіум переможців» 🏆

Три ступені подіуму для переможців гонки!

Подіум — 3 ступені різної висоти з номерами
Крок 1. Ступінь P1 — Box → 25 × 25 × 25 → колір золотий (#F1C40F)
Крок 2. Ступінь P2 — Box → 25 × 25 × 18 → колір срібний (#BDC3C7) → поставити ліворуч від P1
Крок 3. Ступінь P3 — Box → 25 × 25 × 12 → колір бронзовий (#CD6133) → поставити праворуч від P1
Крок 4. Вирівняти — виділи всі 3 → L (Align) → вирівняти по нижньому краю
Крок 5. Текст — Text → "P1" → висота 6 → колір контрастний → підняти на верхівку → Align
Крок 6. Ще текст — так само "P2" і "P3" на відповідних ступенях
Крок 7. Групуємо — Ctrl+A → Ctrl+G
✅ Готово! Подіум переможців! 🥇🥈🥉

⭐⭐⭐⭐ Завдання 5: «Трофей Ф1» 🏆

Кубок переможця гонки!

Крок 1. Основа — Box → 20 × 20 × 3 → колір темно-сірий
Крок 2. Ніжка — Cylinder → діаметр 5, висота 20 → колір золотий → підняти на 3 мм → Align
Крок 3. Чаша — Half Sphere (або Paraboloid) → діаметр 25 → колір золотий → підняти на 23 мм → Align
Крок 4. Виріз чаші — менша Half Sphere → Hole → підняти на 25 мм → Align → Ctrl+G з чашею
Крок 5. Групуємо все — Ctrl+A → Ctrl+G
🏆 Готово! Трофей для переможця!

🎨 Бонус: додай Text "F1" або "2026" на основу трофея
