⭐ Завдання 1 — Світлофор Ф1 🚦

На старті Ф1 — панель з 5 червоними вогнями.

Загоряються по одному → всі гаснуть → СТАРТ!

Будуємо:

  1. Cylinder (стовп) — ⌀5, H=60, сірий
  2. Box (панель) — 50×12×5, чорний, ↑55
  3. Align стовп + панель по центру
  4. Cylinder (вогонь) — ⌀8, H=3, червоний, ↑59
  5. Ctrl+D → зсунути вправо (×4 рази)
  6. Ctrl+A → Ctrl+G
💡 Ctrl+D — копіює об'єкт! Зсунь його вправо і отримаєш 5 однакових вогнів.

⭐⭐ Завдання 2 — Траса з розміткою 🏁

Будуємо:

  1. Box (асфальт) — 120×80×2, темно-сірий
  2. Бордюр: маленький Box 10×5×1, червоний
  3. Ctrl+D → зсунути → білий → Ctrl+D → червоний...
  4. Повторити знизу
  5. Стартова лінія: Box 2×60×0.5, білий
  6. Стартові позиції: Box 5×10×0.5, білий
🏎️ Бордюри (curbs) — це червоно-білі смуги по краях траси. Пілоти їздять по них на поворотах!

⭐⭐⭐ Завдання 3 — Гоночний автомобіль 🏎️

Будуємо:

  1. Box (кузов) — 40×15×6, Radius 2, колір команди
  2. Box (кокпіт) → Hole → Group з кузовом
  3. Cylinder (колесо) — ⌀8, H=4, чорний, повернути 90°
  4. Ctrl+D → розставити 4 колеса
  5. Box (переднє крило) — 20×3×1, спереду
  6. Box (спойлер) — 18×2×8, ззаду, ↑6
  7. Ctrl+A → Ctrl+G

⭐⭐⭐⭐ Завдання 4–5 — Подіум та Трофей 🏆

Подіум переможців:

  • 3 ступені (Box) різної висоти
  • P1 = 25 мм (🥇 золотий)
  • P2 = 18 мм (🥈 срібний)
  • P3 = 12 мм (🥉 бронзовий)
  • Text "P1", "P2", "P3" зверху
🏆 Трофей: Box (основа) + Cylinder (ніжка) + Half Sphere або перевернутий конус (чаша). Колір — золотий!

👀 Робимо світлофор разом!

  1. 1️⃣ Cylinder — діаметр 5, висота 60 → колір сірий
  2. 2️⃣ Box — 50 × 12 × 5 → колір чорний → підняти на 55 мм
  3. 3️⃣ Виділити обидва → L (Align) → центруємо
  4. 4️⃣ Cylinder — діаметр 8, висота 3 → колір червоний → підняти на 59 мм
  5. 5️⃣ Поставити зліва на панелі
  6. 6️⃣ Ctrl+D → зсунути вправо → Ctrl+D → зсунути → повторити (5 вогнів!)
  7. 7️⃣ Ctrl+A → Ctrl+G — готово! 🚦
💡 Після першого Ctrl+D і зсуву — наступні Ctrl+D зсуватимуть на таку ж відстань автоматично!
