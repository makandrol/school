### Завдання 1 (основне): «Світлофор Ф1» ⭐

Стартовий світлофор Формули-1 — 5 червоних вогнів на чорній панелі на стовпі.

**Фігури:**
- Cylinder (стовп): діаметр 5, висота 60, колір сірий (#95A5A6)
- Box (корпус): 50×12×5, колір чорний (#2C3E50), підняти на 55 мм
- Cylinder × 5 (вогні): діаметр 8, висота 3, колір червоний (#E74C3C), підняти на 59 мм

**Кроки:**
1. Cylinder → діаметр 5, висота 60 → колір сірий → це стовп
2. Box → 50×12×5 → колір чорний → підняти на 55 мм (на верхівку стовпа)
3. Align стовп і корпус по центру (по X)
4. Cylinder → діаметр 8, висота 3 → колір червоний → підняти на 59 мм
5. Поставити перший кружечок зліва на корпусі
6. Ctrl+D → зсунути вправо на ~10 мм → повторити 4 рази (5 вогнів)
7. Align все по центру → Ctrl+A → Ctrl+G

### Завдання 2: «Траса з розміткою» ⭐⭐

Ділянка гоночної траси з асфальтом, бордюрами (curbs) і розміткою.

**Фігури:**
- Box (асфальт): 120×80×2, колір темно-сірий (#636E72)
- Box × 2 (бордюри): 120×5×1, колір чергуючий червоно-білий
  - Бордюри — смуги по краях траси. Робимо з маленьких Box: 10×5×1, чергуючи червоний і білий
- Box (стартова лінія): 2×60×0.5, колір білий
- Box (стартові позиції): 5×10×0.5, колір білий, кілька штук на трасі

**Кроки:**
1. Box → 120×80×2 → колір темно-сірий (#636E72) — це асфальт
2. Бордюр (верхній край): маленький Box 10×5×1 → колір червоний → поставити вгорі
3. Ctrl+D → зсунути вправо → колір білий → Ctrl+D → зсунути → червоний → ... (чергуємо)
4. Повторити для нижнього краю
5. Стартова лінія: Box 2×60×0.5 → колір білий → поставити перпендикулярно
6. Стартові позиції: Box 5×10×0.5 → білий → розставити по 2 в ряд

### Завдання 3: «Гоночний автомобіль» ⭐⭐⭐

Спрощений болід Формули-1.

**Фігури:**
- Box (кузов): 40×15×6, заокруглити (Radius 2), колір на вибір
- Box (кабіна/кокпіт): 10×8×5 → Hole → вирізати зверху кузова
- Cylinder × 4 (колеса): діаметр 8, висота 4, повернути на 90° (вздовж X), колір чорний
- Box (переднє крило): 20×3×1, колір як кузов
- Box (заднє крило/спойлер): 18×2×8, тонкий, піднятий на 6 мм

**Кроки:**
1. Box → 40×15×6 → Radius 2 → обрати колір команди (червоний = Ferrari, синій = Red Bull, бірюзовий = Mercedes...)
2. Box (кокпіт) → 10×8×5 → зробити Hole → поставити зверху ближче до центру → Group з кузовом
3. Cylinder → діаметр 8, висота 4 → колір чорний → повернути на 90° по осі X
4. Поставити перше колесо → Ctrl+D → поставити друге → так всі 4
5. Box (переднє крило) → 20×3×1 → поставити спереду, підняти на 1 мм
6. Box (спойлер) → 18×2×8 → поставити ззаду, підняти на 6 мм
7. Ctrl+A → Ctrl+G

### Завдання 4 (творче): «Подіум переможців» ⭐⭐⭐⭐

Три ступені подіуму (1 місце найвище, 2 трохи нижче, 3 ще нижче) + текст "P1", "P2", "P3".

**Фігури:**
- Box × 3 (ступені): різної висоти (P1 = 25 мм, P2 = 18 мм, P3 = 12 мм), ширина 25×25
- Text × 3: "P1", "P2", "P3" на кожному ступені
- Кольори: P1 = золотий, P2 = срібний (#BDC3C7), P3 = бронзовий (#CD6133)
