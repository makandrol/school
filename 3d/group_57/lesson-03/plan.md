# Заняття 3: 🚗 Завершення автомобіля. Дублювання та патерни
**Група:** 5-7 клас
**Дата:** 26.02.2026 (четвер)
**Час:** 15:00–17:20 (3 уроки по 40 хв з перервами)
**Формат:** очно

---

### Мета заняття:
- Завершити модель автомобіля (вікна, фари, бампер)
- Навчитись ефективно використовувати Duplicate (Ctrl+D) та Align (L)
- Створити нові моделі з використанням дублювання та патернів
- Закріпити обертання об'єктів (Rotate)

### Обладнання/інструменти:
- Комп'ютер з доступом до інтернету
- Tinkercad: https://www.tinkercad.com/
- Мишка
- Роздрукований handout для кожного учня

---

### Хід заняття:

#### Урок 1 (15:00–15:40) — Доробляємо автомобіль + нова теорія

- Привітання, інструктаж з БЖД
- Роздати handout із завданнями
- **Нагадування:** що ми робили минулого разу (автомобіль: кузов, кабіна, колеса)
- **Нова тема: Дублювання та патерни**
  - `Ctrl+D` — дублює об'єкт точно в тому ж місці
  - Лайфхак: після дублювання зсунути → наступний `Ctrl+D` повторить той самий зсув!
  - **Приклад:** дублюємо колесо → зсуваємо вправо → ще Ctrl+D → ще колесо на тій самій відстані
  - Це і є **патерн** — повторюваний елемент
- **Нагадування: Align (L)**
  - Виділити кілька об'єктів (Shift+клік) → натиснути L
  - Вирівняти по центру або по краю
- Учні відкривають свої авто та починають Завдання 1 — доробити автомобіль

#### Урок 2 (15:50–16:30) — Нові моделі

- Хто доробив авто → переходять до Завдання 2 — «Робот»
- Це складніша модель: тулуб, голова, руки, ноги, антена
- Акцент на дублюванні парних частин (руки, ноги, очі)
- Хто швидкий → Завдання 3 — «Ракета»
- Вчитель допомагає, показує прийоми точного розміщення

#### Урок 3 (16:40–17:20) — Творче завдання / Закріплення

- Хто не закінчив — доробляє попередні завдання
- Хто все зробив → Завдання 4 — «Космічна станція» (творче)
- Підсумки: нові прийоми (дублювання, патерни, вирівнювання)

---

### Нові інструменти/прийоми Tinkercad:
| Інструмент | Що робить | Де знайти |
|-----------|-----------|-----------|
| Duplicate (Ctrl+D) | Копіювати об'єкт | Виділи → Ctrl+D |
| Duplicate + зсув | Повторює попередній зсув | Ctrl+D → зсунь → Ctrl+D → автоматичний зсув |
| Align (L) | Вирівняти об'єкти | Виділи 2+ → L |
| Mirror (відзеркалення) | Зробити дзеркальну копію | Вручну: Ctrl+D → Flip |

### Очікувані результати:
- Учень вміє: ефективно дублювати об'єкти та створювати патерни
- Учень вміє: вирівнювати кілька об'єктів за допомогою Align
- Учень розуміє: як розбити складну модель на прості частини

### Завдання (на роздатковому аркуші):
1. ⭐ **Автомобіль** — доробити: вікна (Hole), фари (Cylinder), бампер
2. ⭐⭐ **Робот** — тулуб, голова, руки, ноги, антена (дублювання парних частин)
3. ⭐⭐⭐ **Ракета** — корпус, конус-ніс, стабілізатори (обертання + дублювання)
4. ⭐⭐⭐⭐ **Космічна станція** — творче завдання (об'єднай все!)
