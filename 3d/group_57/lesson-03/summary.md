# Заняття 3: 🚗 Завершення автомобіля. Дублювання та патерни
**Дата:** 26.02.2026
**Статус:** 📅 Запланований

## Тема
Завершення моделі автомобіля (вікна, фари, бампер). Дублювання об'єктів (Ctrl+D), патерни, вирівнювання (Align).

## Що вивчили
- Дублювання: Ctrl+D — швидке копіювання об'єкта
- Патерн: Ctrl+D → зсув → Ctrl+D — автоматичний повтор зсуву
- Align (L) — вирівнювання кількох об'єктів по центру/краю
- Парні частини: дублювання + розміщення симетрично (руки, ноги, колеса)
- Hole + Group для вікон в автомобілі

## Що зробили
- ⭐ **Автомобіль** — доробили: вікна (Hole), фари (Cylinder), задні ліхтарі, бампер
- ⭐⭐ **Робот** — тулуб, голова, очі, рот, руки, ноги, антена (парні частини через Ctrl+D)
- ⭐⭐⭐ **Космічний корабель** — корпус (Cylinder), ніс (Cone), ілюмінатор, стабілізатори (патерн по колу)
- ⭐⭐⭐⭐ **Космічна станція** — творче: модулі, сонячні панелі, антена, пристикований космічний корабель

## Інструменти Tinkercad
- Ctrl+D (Duplicate), Align (L), Rotate, Hole, Group
- Патерн: Ctrl+D → зсув → Ctrl+D (автоматичний повтор)
- Box, Cylinder, Sphere, Cone

## Примітки
- Група знає: базові фігури, Hole, Group, Rotate, Move, Scale, Align (базово)
- Тепер також знають: ефективне дублювання, патерни, парні частини
- Автомобіль мав бути завершений з минулого разу — тепер доробили
