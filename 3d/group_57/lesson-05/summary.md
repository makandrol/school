# Заняття 5: 🚀 Космічна подорож
**Дата:** 19.03.2026
**Статус:** ✅ Створено

## Тема
Космічна тематика: ракета (Cone + Cylinder), планета Сатурн (Sphere + кільце з Hole), космонавт (складена модель з багатьох частин), космічна станція (творче).

## Нові концепції
- **Cone** (Basic Shapes): конус — ідеальна форма для носової частини ракети
- **Hole з циліндра для створення кільця**: великий Cylinder − менший Cylinder (Hole) = кільце
- **Складання моделі з багатьох деталей**: космонавт має 8+ частин, важливо правильно позиціонувати

## Що зробили
- ⭐ **Ракета** — Cylinder (корпус, ⌀20, H=50) + Cone (носова частина, ⌀20, H=20, підняти на 50) + Box (стабілізатори, 3×10×15, ×3 через Ctrl+D, повернуті на 120°) + Sphere (вікно, блакитний)
- ⭐⭐ **Планета Сатурн** — Sphere (⌀40, золотий) + кільце (Cylinder ⌀70, H=2 − Cylinder ⌀42 Hole) + Sphere (місяці, ×2)
- ⭐⭐⭐ **Космонавт** — Sphere (шолом, ⌀20) + Box (тіло, 18×14×25) + Cylinder (руки, ⌀6×22, ×2) + Cylinder (ноги, ⌀7×20, ×2) + Box (рюкзак, 14×8×20) + Half Sphere (забрало, золотий)
- ⭐⭐⭐⭐ **Космічна станція** — творче: циліндричні модулі + Box сонячні панелі + антена + з'єднання

## Закріплені навички
- Hole + Group для вирізання (кільце Сатурна)
- Align для позиціонування
- Duplicate / Ctrl+D (стабілізатори, місяці, руки/ноги)
- Кольори та піднімання об'єктів
- НОВЕ: Cone (конус) як фігура
- НОВЕ: Створення кільця через Hole з циліндра
- НОВЕ: Складна модель з 8+ деталей (космонавт)
