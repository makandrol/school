  • 🚀 Перша людина в космосі — Юрій Гагарін (1961)
  • 🌙 Перша людина на Місяці — Ніл Армстронг (1969)

Сьогодні ми будемо моделювати власний космічний світ! 🌌

📋 Завдання на сьогодні

⭐РакетаКорпус + носовий конус + стабілізатори + вікно
⭐⭐Планета СатурнСфера + кільце + маленькі місяці
⭐⭐⭐КосмонавтШолом + тіло + руки + ноги + рюкзак
⭐⭐⭐⭐Космічна станціяМодулі + сонячні панелі + антена
📄 Детальна інструкція — у роздатковому матеріалі!

⭐ Завдання 1 — Ракета 🚀

Будуємо:

  1. Cylinder (корпус) — ⌀20, H=50, білий
  2. Cone (носова частина) — ⌀20, H=20, червоний → підняти на 50
  3. Align конус + циліндр по центру
  4. Box (стабілізатор) — 3×10×15, червоний → 3 штуки навколо
  5. Sphere → Hole (вікно) → Group з корпусом
💡 Cone знаходиться в Basic Shapes. Він ідеально підходить для носової частини ракети!

⭐⭐ Завдання 2 — Планета Сатурн 🪐

Будуємо:

  1. Sphere (планета) — ⌀40, золотисто-жовтий
  2. Cylinder (кільце) — ⌀70, H=2, помаранчевий
  3. Cylinder → Hole — ⌀42, H=3 → Group з кільцем
  4. Align планету + кільце
  5. Sphere (місяць) — ⌀8, сірий → поставити збоку
🪐 У Сатурна 146 місяців — більше ніж у будь-якої іншої планети!

⭐⭐⭐ Завдання 3 — Космонавт 👨‍🚀

Будуємо:

  1. Sphere (шолом) — ⌀20, білий
  2. Box + Radius (тіло) — 18×14×25, білий, ↑піднімаємо
  3. Cylinder (рука) — ⌀6, H=22, білий × 2
  4. Cylinder (нога) — ⌀7, H=20, білий × 2
  5. Box (рюкзак) — 14×8×20, сірий, ззаду
  6. Half Sphere (забрало) — ⌀14, золотий → на шолом
  7. Ctrl+A → Ctrl+G

⭐⭐⭐⭐ Завдання 4 — Космічна станція 🛸

Ідея:

  • Cylinder (модулі) — різних розмірів, з'єднані між собою
  • Box (сонячні панелі) — плоскі, сині, по боках
  • Cylinder (антена) — тонка, зверху

Це творче завдання — побудуй станцію на свій смак!

🏆 Покажи вчителю — найкращі роботи покажемо всьому класу!

👀 Робимо ракету разом!

  1. 1️⃣ Cylinder — діаметр 20, висота 50 → колір білий
  2. 2️⃣ Cone (Basic Shapes) — діаметр 20, висота 20 → колір червоний
  3. 3️⃣ Підняти конус: чорний конус ↑ → 50 мм
  4. 4️⃣ Виділити обидва → L (Align) → центруємо
  5. 5️⃣ Box — 3 × 10 × 15 → колір червоний → поставити збоку внизу
  6. 6️⃣ Ctrl+D → повернути на 120° → Ctrl+D → ще на 120° (3 стабілізатори!)
  7. 7️⃣ Sphere — діаметр 10 → Hole → поставити на корпус (вікно!) → Ctrl+G
  8. 8️⃣ Ctrl+A → Ctrl+G — готово! 🚀
💡 Для стабілізаторів: поверни кожен на 120° (Shift утримуй для повороту по 45°, або впиши число вручну)
