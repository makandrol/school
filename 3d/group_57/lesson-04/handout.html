⭐ Завдання 1: «Іменний кубик»

Куб з гравірованим ім'ям на одній грані та опуклим текстом на іншій.

Іменний кубик — куб з гравірованим ім'ям
Крок 1. Куб — Box 40×40×40 → вибери колір (наприклад, синій #3498DB)
Крок 2. Текст — Text → введи своє ім'я → Height: 10 → Font: Sans
Крок 3. Гравірування — клікни на текст → зроби Hole (інспектор → Hole)
Крок 4. Позиція — поверни текст вертикально (обертання 90° по X) → постав на передню грань куба
Крок 5. «Втопи» — зсунь текст на 1-2 мм всередину куба
Крок 6. Вирізай — виділи куб + текст → Ctrl+G → текст вирізається!
💡 Якщо хочеш опуклий текст — не роби Hole, а підніми Text на 1 мм над гранню і згрупуй.
✅ Готово! Іменний кубик створено! 🎲

🎨 Бонус: додай символ або декор на іншу грань. Спробуй різні шрифти (Serif, Mono).

⭐⭐ Завдання 2: «Трофей»

Кубок на підставці з гравірованим написом "ПЕРЕМОЖЕЦЬ" або своїм ім'ям.

Трофей — кубок + підставка + напис
Крок 1. Чаша — Cylinder d=40, h=30 → зверху Cylinder d=34, h=28 як Hole → Ctrl+G (чаша!)
Крок 2. Ніжка — Cylinder d=8, h=25 → Align по центру чаші → опустити
Крок 3. Підставка — Box 50×50×8 → колір темний → Align
Крок 4. Напис — Text → "ПЕРЕМОЖЕЦЬ" або "#1" → Height 5 → Hole → помістити на підставку → Ctrl+G
Крок 5. Кольори — чаша: золотий (#F1C40F), підставка: темний (#2C3E50), ніжка: темно-золотий
Крок 6. Групуємо все — Ctrl+A → Ctrl+G
✅ Готово! Трофей створено! 🏆

🎨 Бонус: додай ручки (тонкі Cylinder або Torus) та зірку зверху.

⭐⭐⭐ Завдання 3: «Смартфон»

Моделюй реалістичний смартфон: корпус, екран, камера, кнопки, текст бренду.

Смартфон — корпус + екран (Hole) + деталі
Крок 1. Корпус — Box 35×70×4, Radius: 3 (заокруглення) → колір темний
Крок 2. Екран — Box 30×58×3 → Hole → Align по центру → трохи підняти → Ctrl+G (вирізаємо)
Крок 3. Скло — Box 30×58×1 → Transparent → помістити в екран
Крок 4. Камера — Cylinder d=4, h=1 → зверху → колір сірий
Крок 5. Бренд — Text → назва твого бренду → Height 3 → гравірувати на задній панелі
Крок 6. Кнопки — маленькі Box на бічних гранях (гучність, живлення)
✅ Готово! Смартфон створено! 📱

🎨 Бонус: додай чохол (трохи більший Box, Hole всередині), додай «іконки» на екран (маленькі кольорові Box).

⭐⭐⭐⭐ Завдання 4: «Дизайн кімнати»

Створи 3D-модель кімнати з меблями. Мінімум 3 предмети!

Приклад кімнати — стіни + ліжко + стіл + вікно

Що потрібно зробити:

  • Підлога — Box (велика плоска площина, наприклад 100×80×1)
  • 2 стіни — Box (тонкі, високі, по краях підлоги)
  • Мінімум 3 предмети меблів на твій вибір
  • Вікно — Box → Hole в стіні (або прозоре скло)

Ідеї для меблів:

  • 🛏️ Ліжко — Box (матрац) + Box (спинка) + подушка (Box менша)
  • 🪑 Стілець — Box (сидіння) + 4 Cylinder (ніжки) + Box (спинка)
  • 🖥️ Стіл з комп'ютером — Box (стільниця) + ніжки + Box (монітор)
  • 📚 Книжкова полиця — Box (каркас) + маленькі Box (книги, різні кольори)
  • 🪟 Вікно — Box → Hole в стіні → Transparent Box (скло)
💡 Використовуй Ctrl+D для дублювання однакових предметів (ніжки стільця, книги).
🏆 Покажи вчителю — найкращі кімнати покажемо всьому класу!
