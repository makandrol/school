# Заняття 4: 🔤 3D-текст, гравірування та дизайн
**Дата:** 05.03.2026
**Статус:** 📅 Запланований

## Тема
3D-текст (Text shape), гравірування (Text → Hole), кольори та HEX-коди, прозорість. Дизайнерські проєкти.

## Нові концепції
- **3D-текст:** Basic Shapes → Text → Inspector: Text, Height, Font, Bevel
- **Гравірування:** Text → Hole → "втопити" на 1-2 мм в об'єкт → Ctrl+G = вдавлений напис
- **Опуклий текст:** Text (Solid) → підняти на поверхню → Ctrl+G = виступаючий напис
- **Кольори / HEX:** палітра + введення HEX-коду (#FF5733)
- **Прозорість:** Transparent для скла/вікон

## Що зробили
- ⭐ **Іменний кубик** — Box 40×40×40 + гравіроване ім'я (Text → Hole на грані)
- ⭐⭐ **Трофей** — чаша (Cylinder - Hole) + ніжка + підставка + гравірований напис
- ⭐⭐⭐ **Смартфон** — корпус (Box з Radius) + екран (Hole) + скло (Transparent) + камера + бренд (гравірування)
- ⭐⭐⭐⭐ **Дизайн кімнати** — підлога + 2 стіни + мінімум 3 предмети меблів

## Закріплені навички
- Hole + Group (повторення)
- Align, Duplicate, Rotate (повторення)
- НОВЕ: 3D-текст (Text shape) з різними шрифтами
- НОВЕ: Гравірування (Text → Hole)
- НОВЕ: Кольори та HEX-коди
- НОВЕ: Прозорість (Transparent)
