### Хід заняття:

#### Урок 1 (15:00–15:40) — Теорія + демонстрація

- **Кольори:** палітра кольорів, HEX-коди, прозорість (Transparent)

- Демонстрація: вчитель робить іменний кубик
- Учні починають Завдання 1

#### Урок 3 (16:40–17:20) — Творче завдання

### Нові інструменти:

**3D-текст:**
- Basic Shapes → прокрутити → "Text"
- Inspector: Text (зміст), Height (висота), Font (шрифт: Sans, Serif, Mono), Bevel (фаска)
- Segments — якість (більше = гладші літери, але важчі)

**Гравірування (вдавлений текст):**
1. Створити Text → зробити Hole
2. Помістити на поверхню об'єкта (опустити на 1-2 мм всередину)
3. Виділити обидва → Ctrl+G → текст "вирізається" з об'єкта

**Кольори:**
- Клікнути на об'єкт → палітра справа
- Можна ввести HEX-код (#FF5733)
- Transparent — напівпрозорий (для скла)

---

### Завдання 1 (основне): «Іменний кубик» ⭐
Куб з ім'ям на одній грані (гравірування) + колір

### Завдання 2: «Трофей / Кубок» ⭐⭐
Кубок (з минулого уроку чашка) + підставка + напис гравірований

### Завдання 3: «Смартфон» ⭐⭐⭐
Box + заокруглення + екран (Hole) + кнопки + текст бренду

### Завдання 4 (творче): «Дизайн кімнати» ⭐⭐⭐⭐
Стіни + підлога + меблі (ліжко/стіл/стілець) мінімум 3 предмети
