# Шаблон роздаткового матеріалу (3D-моделювання, Tinkercad)

Цей файл друкується та роздається дітям. Формат: чіткий, без зайвих прикрас, зручний для друку на А4.

**⚡ ВАЖЛИВО:** Між кожним завданням ОБОВ'ЯЗКОВО ставити:
```html
<div style="page-break-after: always;"></div>
```
Це забезпечує розрив сторінки при друку — кожне завдання на окремому аркуші.

**⚡ ВАЖЛИВО:** Після створення файлу запустити `bash ./build_all.sh path/to/lesson/` для генерації HTML.

## Правила для роздаткового матеріалу:
- **Кожне завдання на ОКРЕМІЙ сторінці** — ставити `<div style="page-break-after: always;"></div>` між ними
- **Перша сторінка** — заголовок + нагадування нових інструментів (коротко, БЕЗ повної інструкції)
- **Завдання 1** — обов'язкове для всіх, з ДЕТАЛЬНОЮ покроковою інструкцією
- **Завдання 2** — додаткове, для тих хто зробив перше; інструкція трохи менш детальна
- **Завдання 3+** — для найшвидших; лише підказки, заохочення до самостійності
- Мінімум 2 завдання, бажано 3-4
- Для 3-4 класу: дуже детальні кроки в Завданні 1 (буквально кожна дія)
- Для 5-7 класу: можна менш детально, більше самостійності
- Розміри вказувати конкретно (наприклад: "Box: ширина 20, глибина 20, висота 30")
- Формат: чорно-білий, зручний для друку, без кольорових елементів
- Використовувати зірочки (⭐) для позначення рівня складності
